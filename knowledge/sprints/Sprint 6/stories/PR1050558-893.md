## Description

As a Recruiter, I need to request approval for clinician compensation, so that I can offer appropriate pay and incentives for clinicians to join SCP

## Acceptance Criteria

Given that I am a Recruiter in the Corporate Support apps

When I navigate to a **Clinician** record

Then I can click on the **Contract Request** button 

And the Facility Selection screen is displayed:

- If I have already initiated a Contract Request (contract record where Status = Draft)

** Then I can select between “Continue with Draft Request” or “Discard and Start New Request”

- If I have not initiated a Contract Request

*** I can select from a list of Healthcare Practitioner Facility records for that clinician where **Status = Prospect, Candidate, Waiting to be Scheduled, Active, WTBS Here-WRKG at Sister Site* to determine which HPFs I want to draft the Contract Request for

*** Fields to display: Account, Facility Title, Status, Contracting Status

*** User must select at least 1 facility to proceed

*** If the clinician does not have Healthcare Practitioner Facility records where **Status = Prospect, Candidate, Waiting to be Scheduled, Active, WTBS Here-WRKG at Sister Site*, display the following message “This clinician does not have a Healthcare Practitioner Facility record. Please complete the Create HPF workflow.”

*** Process ends; user can select Finish and the guided experience closes. 

- Navigation buttons

** Cancel

*** Returns to Clinician record and no records were created or updated
