## Solution

Given that we click on the Contract Request button on Clinician, Omniscrip Clinician/ContractRequest runs.

1. Screen Shows Employment Type Radio Button using Picklist (Contract.Employment Type)  Built Part of  [https://salesforce.atlassian.net/browse/PR10505581332|https://salesforce.atlassian.net/browse/PR10505581332|smart-link] 

1. Based on selected Employment Type query Contract record the Account Id = Clinician Id and Record Type= Clinician and Employment Type = Selected Type from Step1” and Status = Draft and sort by Last Modified Desc and pick the first record

## If a Contract Record exists then display radio button (Continue with Draft, Discard and Start new)

### Continue with Draft is selected

#### Navigate to Contract Record Page With Summary Tab

### Discard and Start new is selected

#### Delete Draft Contract and all child records Provider Network Contract, Contract Payment Agreement, Preventive Care Agreement

#### Go to Step 2.b

## If a Draft Contract Record did't exists 

### query Contract record the Account Id = Clinician Id and Record Type= Clinician and Employment Type = Selected Type from Step1” and Status = Executed 

#### Clone the Contract record 

##### Name: Clinician Name + Todays Date

##### Parent Contract (ParentContractId PreviousContractId) = Contract from where data is Cloned from

##### Status = Draft

#### Clone new Provider Network Contract records with all Active HPF Part of the Contract Record and Deleted? = False (what is a Active HPF  inside a Contract?)

#### Clone new Contract Payment Agreement records 

##### Provider Network Contract = new Provider Network Contract created in Step (ii)

##### Cloned From = Original Contract Payment Agreement

#### Clone new Preventive Care Agreement records with the relation to new Provider Network Contract lookup

##### Cloned From =  Preventive Care Agreement record that is cloned from

##### Facility Rate = Facility Rate, Facility Title=Facility Title, Shift=Shift, Type=Type, Rate=Rate, Unit of Measure=Unit of Measure, Time Type=Time Type, Override Prohibited= Override Prohibited **(all fields need to be copied)**
