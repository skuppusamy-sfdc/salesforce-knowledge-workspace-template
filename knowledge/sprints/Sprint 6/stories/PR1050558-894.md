## Description

As a Recruiter, I need to request approval for clinician compensation, so that I can offer appropriate pay and incentives for clinicians to join SCP

## Acceptance Criteria

Given that I am a Recruiter in the Corporate Support apps

When I navigate to a Clinician **Contract** record (user successfully completed Contract Request guided experience [https://salesforce.atlassian.net/browse/PR10505581070|https://salesforce.atlassian.net/browse/PR10505581070|smart-link] )

Then on the **Summary tab**:

** The user has the ability click the “Add HPF” button to add additional Healthcare Practitioner Facilities if contract **status = Draft* (covered under story [https://salesforce.atlassian.net/browse/PR10505581389|https://salesforce.atlassian.net/browse/PR10505581389|smart-link] )

- User has the ability to see all Healthcare Practitioner Facility (HPF) records that have been linked to the contract

- Within each Healthcare Practitioner Facility:

** The user has the ability see configured clinician rates for that particular facility

- The Rates are:

** Group by:

*** Type of Rate (shift, stipend, variable, etc)

**** Displayed in collapsible sections (collapsed by default)

*** Type (Shift Full Name or Type)

*** Rate and Unit of Measure (display “$” before Rate)

** The user has the ability to click the “Configure Rates” button to add more rates if contract **status = Draft*

- The user has the ability to Delete:

** The user can delete at a facility level

*** The system shall prompt the user to confirm deletion

*** The system shall remove the HPF from this contract and delete the PNC and all related rate records 

*** The system shall prevent the user from deleting all HPFs, a contract must have at least one HPF

** The user can delete at a Type of Rate (shift, stipend, variable, etc)

*** The system shall prompt the user to confirm deletion

*** The system shall delete all records for that Type of Rate

** The user can delete at a specific Rate level (Rate & Unit of Measure)

*** The system shall prompt the user to confirm deletion

*** The system shall delete that single rate record

Note: Configure Rates button are part of stories:
