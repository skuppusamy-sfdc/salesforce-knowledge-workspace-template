## Description

As a Recruiter, I need Salesforce to maintain a single record for a clinician’s primary taxonomy and primary specialty, so that I have accurate data maintained for clinicians

## Acceptance Criteria

Given I am a Recruiter

When I create or update a Healthcare Provider Specialty so that Primary Board Specialty = checked

Then all other Healthcare Provider Specialty records for that clinician are updated so that Primary Board Specialty = unchecked

Given I am a Recruiter

When I create or update a Healthcare Provider Taxonomy so that Primary Taxonomy = checked

Then all other Healthcare Provider Taxonomy records for that clinician are updated so that Primary Taxonomy = unchecked

## Solution

For both Taxonomy and Specialty within the following guided experiences:

- Update NPI -  update existing/create new record as primary & uncheck primary from all other records. Do not delete taxonomy records.

- Create HPF -  update existing/create new record as primary & uncheck primary from all other records. Do not delete taxonomy records.

- Create Clinician - create new record as primary
