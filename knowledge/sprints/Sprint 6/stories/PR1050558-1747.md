## Description

As an Integration User, I need two new formula fields on Provider Network Contract object, so that I locate the correct records.

## Acceptance Criteria

Given I am a System Administrator in Core HC

When I am on the Provider Network Contract object

Then the 2 **new formula fields** are available to the Integration User as specified in the Data Dictionary

- Facility Title

- Time Type

And update the Provider Network Contract **layout** by replacing **the old picklist fields** with the new **new formula fields.**

## Solution

- Ensure when you create the new fields that visibility is granted to all permission sets with access to the object. 

Create 2 formula fields that points to the respective values of Facility_Title__r.Name and Time_Type__c from HPF. Time_Type__c is a picklist value so make sure to use the TEXT() formula.

Update the LRP for Provider Network Contract (there is just one) to use the new formula fields instead of the old ones.
