## Description

As a System Administrator, I need clear list views on the Code Set object, so that I can clearly navigate between different types of records

## Acceptance Criteria

Given I am a System Administrator in the Corporate Support apps

When I navigate to the Code Set tab

Then I am able to see the following list views:

- All Code Sets

** Columns to display: Name, Code Set Type, Code, Active (use Active__c formula field), Last Modified Date

** Filters: none

- Facility Titles

** Columns to display: Name, Effective Date, End Date,  Active (use Active__c formula field), Last Modified Date, Code Set Sub Type, Can Have Leadership Incentive?

** Filters: Code Set Type = Facility Title

- Risk Review Questions

** Columns to display: Name, Effective Date, End Date, Active (use Active__c formula field), Last Modified Date

** Filters: Code Set Type = Provider Risk Review

- Rate Configurations

** Columns to display: Name, Code Set Type, Effective Date, End Date, Active (use Active__c formula field), Last Modified Date

** Filters: Code Set Type = Shift, Bonus, Stipend, Variable or Leadership Incentives

- Recently Viewed (standard list view)
