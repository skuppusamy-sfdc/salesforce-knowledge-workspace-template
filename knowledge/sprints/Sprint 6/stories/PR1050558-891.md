## Description

As a Prospective Clinician, I need to view information about my open applications in HC portal, so that I can understand the latest status and see important information about the job

## Acceptance Criteria

Given I am a Clinician in the Prospect Portal

If I have an ATS Applicant record where Status = Active

Then I am able to see the following: 

- My Applications

** List of ATS Applicant records where Applicant = my person account record 

** And when I click the Job field, then I can see the following: 

**** **Withdraw Application button*

Given I am a Clinician in the Prospect Portal

If I have an ATS Applicant record where Status = Active

And I click the Withdraw Application button

Then the following happens

- Confirmation Screen

** User can see the following message “Are you sure you want to withdraw your application to [Job Name]? This action cannot be undone.”

*** User is required to select **Withdrawal Reason* (single select picklist)

*** If Other is selected, user must populate Other Reason

** Navigation Options

*** Cancel - close out the window, no updates made

*** Next with checking the checkbox - the following updates are made: 

- Automated Updates

** ATS Applicant record 

**** Status = **Rejected*

**** Declined Reason = **Candidate Declined*

*** Declined Detail = [what user selected]

*** Other Reason = [what user populated]

** Healthcare Practitioner Facility record (where Practitioner Facility Name = Account Name from Job record AND Practitioner = Applicant from ATS Applicant record)

**** Status = **Inactive*

NOTE: if in the event the update fails then a message will display with a finish button: There was an error while processing the request, please contact the system administrator.

----

Withdrawal Reason

- Accepted Another Position

- Benefits Package Not Competitive

- Clinical Scope Not Aligned with My Skills or Interests

- Compensation Package Not Competitive

- Concerns about Practice Culture, Stability, or Reputation

- Contract Terms (Non-Financial) Not Acceptable

- Credentialing Process Too Burdensome

- Decided to Stay at Current Position

- Licensing or Visa Complications

- Location Not Desirable

- Negative Interview Experience

- No Longer Seeking a New Opportunity

- Organization’s Values Not Aligned with Mine

- Personal Circumstances Have Changed

- Poor Communication During Recruitment

- Practice Model Not a Good Fit

- Prefer Not to Disclose

- Process Taking Too Long

- Scheduling Expectations Not Acceptable

- Other

## Solution

- Use a flex card to display each ATS Applicant record. 

** For each ATS Applicant record that should be a button **Withdraw Application*

- When the user clicks on the Individual button show the message as specific in AC

- Update the ATS record with the details specific in AC

- Security Matrix updates:

** Tab = Permission Set

*** Permission Set = Portal User

**** Add Edit access to the following object:

****** **ATS Applicant*

**** Add Read & Edit access to the following object:

****** **Healthcare Practitioner Facility*
