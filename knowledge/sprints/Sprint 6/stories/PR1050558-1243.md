## Description

As a payroll team member, I need pay code data to be available in Salesforce, so that I can reduce manual entry between systems

## Acceptance Criteria

Given I am a SFO in the Corporate Support apps

When I navigate to a Client Contract record where Status = New

Then on tab = Rate Pay Codes, I can click Add Pay Codes from available Shift records

And I am able to select multiple values from records on the Pay Code object

## Solution

Create Rate Pay Codes Tags Tab in Contract of Record Type = Client

Display All the **Preventive Care Agreement** records for the Contract where **Contract Payment Agreement** Type = Shift

- Field to Display

*** **Shift*.Shift Full Name

*** **Facility Title.*Name

When User clicks the Add Pay Codes a multi select picklist is displayed and that shows all data from **Pay Code** where Active = checked

The selected Pay Code from the multi select added to **Rate Pay Code**

**Rate Pay Code**.Preventive Care Agreement = **Preventive Care Agreement**.Id

**Rate Pay Code**.Pay Code = **Pay Code**.Id

When the User delete the individual record delete **Rate Pay Code** record.

Display **Rate Pay Code**.Pay Code Name in the UI
