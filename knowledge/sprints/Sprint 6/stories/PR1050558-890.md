## Description

As a Prospective Clinician, I need to view information about my open applications in HC portal, so that I can understand the latest status and see important information about the job

## Acceptance Criteria

Given I am a Clinician in the Prospect Portal

If I have an ATS Applicant record where Status = Active

Then I am able to see the following: 

- My Applications [new tab]

** List of ATS Applicant records where Applicant = my person account record

*** Fields to display: **Job, Initial Application Date, Recruiter*

*** Do we need to adjust the ATS Stage values? Add a formula field called Applicant Status? If Stage = Offer, show Evaluating Application

** And when I click [expand] the Job field, then I can see the following: 

**** **Job Name, Facility Name, Recruiter, Recruiter Phone (text-enabled), Recruiter Email*

## Solution

Use Experience Cloud Sharing Set to share ATS Applicant and Job Request 

Create a Flex Card to Display the ATS Applicant Record(s)

- Parent Record ID = person account for the logged in user

** Job Name = bpats__Job__c.Name

** Application Date = bpats__Initial_Application_Date__c

** Recruiter = bpats__Job__r.Recruiter__c.Name

** Stage = bpats__Stage__c

** Facility Name = bpats__Job__r.Facility_Name__c.Name

** Recruiter Phone = bpats__Job__r.Recruiter__c.Phone

** Recruiter Email = bpats__Job__r.Recruiter__c.Phone
