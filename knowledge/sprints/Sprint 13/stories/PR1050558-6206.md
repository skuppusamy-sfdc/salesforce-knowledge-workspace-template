## Description

**Steps to reproduce:**

1. Open a clinician record with Intake form case in Not Started status and no assessments are created yet

1. Login to Experience cloud as the clinician

1. User should be able to view Clinician profile updates and other forms

1. Submit the Clinician profile updates form first by updating Name and NPI fields

1. Verify if the tasks for Legal Name and NPI change are created

1. Submit the other forms available

1. Verify if the intake form case is closed

**Expected Result:**

Legal Name and NPI change tasks should be created as soon as Clinician profile updates form is submitted

**Actual Result:**

Legal Name and NPI change tasks are created after intake form case is closed and not immediately after Clinician profile updates form is submitted

## Root Cause

Tasks are created by the scheduled batch job which takes some time to complete
