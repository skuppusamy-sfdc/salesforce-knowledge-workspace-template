**Description:** 

Please add these 3 text fields to Integration_Log__c object in SF. We will start leveraging the category field immediately grouping logs for reporting and retries. We will also have long term ability to move Globals to dedicated fields and free up CorrelationID and Related_SF_Record to store SF ids for all error types.

  Error_Category__c           (Text Area 255)

  Global_Provider_Id__c     (Text Area 255)

  Global_Facility_Id__c       (Text Area 255)

**Business Impact:**

Troubleshooting is more difficult because fields are used inconsistently and there is no way to search logs from within SF

Currently over 120K logs and no designation
