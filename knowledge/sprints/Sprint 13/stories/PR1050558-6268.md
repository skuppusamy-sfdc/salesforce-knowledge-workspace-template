## Description

**Steps to reproduce:**

1. Login as System Administrator

1. Open a clinician account having assessment records created within last 180 days

1. Click ‘Intake Form’ button from Highlights panel

1. Verify if an intake form case is created

1. Login to Experience Cloud as the clinician

1. Verify if no new assessment records are created for the ones that were created within last 180 days

**Expected Result:**

1. No new assessment records should be created if there are existing assessment records created within last 180 days

1. Intake form case status should be updated as ‘Closed’ on creation if there are no new assessment records to be created

**Actual Result:**

1. New assessment records are created even though there are existing assessment records created within last 180 days

1. Intake form case status is not ‘Closed’  (dependent on #1)

## Root Cause

AC was not implemented
