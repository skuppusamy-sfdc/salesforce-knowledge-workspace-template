Not all Contact goes for Approval so the original story solution need to change to 

When the User **Submit the Contract  (Submit Contract Button)**

if PreviousContract.Employment_Type__c = Same as Current Contract then Set  **Term Sheet Status** = **‘Not Applicable’ (only for EE and IC)**

if PreviousContract.Employment_Type__c = “Employee” and the new Contract.Employment_Type__c = “Independent Contractor” then Set  **Term Sheet Status** = **‘Not Applicable’**
