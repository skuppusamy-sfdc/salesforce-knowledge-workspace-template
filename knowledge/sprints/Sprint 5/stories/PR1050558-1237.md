## Description

Issue  The Account field is missing on the Parent and Child Case of Record Type  Clinician Onboarding Facility

Expected Result  The Account field should be present on the Parent and Child Case of Record Type  Clinician Onboarding Facility.

Actual Result  The Account field is missing on the Parent and Child Case of Record Type  Clinician Onboarding Facility

Steps to Reproduce:

1-Login as MSL USER and login into the Core HC Org.

2-Open a Account record of record type Clinician.

3-Ensure HPF Record is present with Status Prospect and no  open cases of Record type = Clinician Onboarding & Type = Facility.

4-Ensure Credentialing Status is not equals to "Active","Pending", "Scheduled Pending", "Temporarily Inactive" .

5-Initiate the Initial Early Onboarding flow and complete it.

6-Navigate to the Parent and Child cases which got created.

7-Verify Account field is not displayed.

The Data dictionary got updated after the Deployment was done, hence a requirement change after deployement.
