## Description

As a Recruiter, I need to request approval for clinician compensation, so that I can offer appropriate pay and incentives for clinicians to join SCP

## Solution

In OmniScript built in user story [https://salesforce.atlassian.net/browse/PR1050558893|https://salesforce.atlassian.net/browse/PR1050558893|smart-link] , add a step after facility selection. 

This step should contain customOmniSelect LWC with types of rates as values, so create an Integration Procedure and Data Mapper to retieve and pass the values to the LWC. See following Data Mapper example to retrieve Type of Rates (make sure values are not hardcoded:
