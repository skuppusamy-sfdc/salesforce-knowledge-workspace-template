## Description

**Steps to reproduce:**

1. Log in to Salesforce Core HC org as Recruiter

1. Navigate to Corporate Support app and open a Clinician account record matching having **Open Onboarding** case record

1. Navigate to Cases tab and open any record from **'Open Onboarding'** related list

1. Verify if the fields are displayed on the Details tab as per data dictionary

**Expected Results:**

All  fields should be displayed on the Details tab as per data dictionary

**Actual Results:**

'Comments' field is missing on Clinician Onboarding case page layout for Recruiter

## Root Cause

Permission was missed for this field. As it was already existing field, so it was an understanding that permission for that field will also be there in qa but it was not. As this story was related to pagelayout/LRP comments field was not created, hence no permission set was this field was initially deployed. 

Later on the required permission set got deployed.
