## Description

As a Medical Staff Liaison, Enrollment Specialist, Data Quality team member, I need the system to notify me if there is already an SCP Onboarding case for Clinician, so that I do not need to proceed with the Initiate Early Onboarding process.

## Acceptance Criteria

Given that I am a Medical Staff Liaison, Enrollment Specialist, Data Quality team member in the Corporate Support App

When I click “Initiate Early Onboarding” on a clinician record 

And the Clinician already has an open SCP Onboarding Case

And I select SCP 

Then I can see the following error message “**This clinician has an open SCP Onboarding case. Please review existing onboarding case.**”

And have the ability to exit the guided process.

Given that I am a Medical Staff Liaison, Enrollment Specialist, Data Quality team member in the Corporate Support App

When I click “Initiate Early Onboarding” on a clinician record 

And the Clinician already has an open SCP Onboarding Case

And I select Both

Then I can see the following error message “**This clinician has an open SCP Onboarding case. Please review existing onboarding case.**”

And I can see the Facilities or the defined error message when no Healthcare Practitioner facilities are found in story [https://salesforce.atlassian.net/browse/PR1050558822|https://salesforce.atlassian.net/browse/PR1050558822|smart-link]  

And 

 - if no available facilities then allow the user to exit the guided process.

 - If there are available facilities, proceed with current process

Given that I am a Medical Staff Liaison, Enrollment Specialist, Data Quality team member in the Corporate Support App

When I click “Initiate Early Onboarding” on a clinician record

And I select Facility Specific, SCP, or Both

**Then Priority is required and cannot be blank**

And upon clicking Finish, the system should direct the user back to the Clinician record. 

And Cases created with Priority selected by user  covered under: [https://salesforce.atlassian.net/browse/PR1050558562|https://salesforce.atlassian.net/browse/PR1050558562|smartlink] , [https://salesforce.atlassian.net/browse/PR1050558622|https://salesforce.atlassian.net/browse/PR1050558622|smartlink] , [https://salesforce.atlassian.net/browse/PR1050558610|https://salesforce.atlassian.net/browse/PR1050558610|smartlink] , [https://salesforce.atlassian.net/browse/PR1050558621|https://salesforce.atlassian.net/browse/PR1050558621|smart-link]
