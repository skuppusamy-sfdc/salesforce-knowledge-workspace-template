## Description

As a Data Quality team member, I need to create a “Contract” for leased and locums clinicians, so that we can move clinicians from initial data entry to onboarding

## Acceptance Criteria

Given I am a Data Quality team member in the Corporate Support apps

When I navigate to a clinician record

Then I am able to select the Contract Request button in the highlights panel

And the following happens:

- Facility Selection screen built in [https://salesforce.atlassian.net/browse/PR1050558893|https://salesforce.atlassian.net/browse/PR1050558893|smart-link] 

- Skip Contract Confirmation

** If selected Healthcare Practitioner Facility record(s) have a Facility Title of External Locums, Leased Employee, Leased Medical Director, Managed NP/PA, Managed Nurse, Managed Physician

**** Display the following message: “A signed **SCP* contract is not needed for this clinician. Please select next to confirm that this clinician is ready to start onboarding.”

*** Confirm or Update: Effective From for each selected HPF.

**** Effective From is pre-populated with: existing value or today’s date if blank

** Navigation buttons

*** Cancel

**** Returns to Clinician record and no records were created or updated

*** Next
