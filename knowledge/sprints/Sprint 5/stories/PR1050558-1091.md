## Description

As a Recruiter, I need to request approval for clinician compensation, so that I can offer appropriate pay and incentives for clinicians to join SCP

## Acceptance Criteria

_Story focus: implementing the functionality on the Facility Selection screen when user selects_ “Continue with Draft Request” or _“Discard and Start New Request”.  Story 893 created the UI for the 2 selection options._ 

Given that I am a Recruiter in the Corporate Support apps

When I navigate to a **Clinician** record

And I can click on the **Contract Request** button 

And the Facility Selection screen is displayed:

- If I have already initiated a Contract Request (contract record where Status = Draft)

** Then I can select between “Continue with Draft Request” or “Discard and Start New Request”

And if the user selected “**Continue with Draft Request**”

Then the user can select from a list of Healthcare Practitioner Facility records for that clinician where Status = Prospect, Candidate, Waiting to be Scheduled, Active, WTBS Here-WRKG at Sister Site.

And previously selected Healthcare Practitioner Facility are selected by default 

- Navigation buttons

** Cancel

*** Returns to Clinician record and no records were created or updated
