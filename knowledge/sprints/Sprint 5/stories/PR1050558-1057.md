## Description

Steps to Reproduce:

1-Login as SFO in the Core HC Org.

2-Navigate to Corporate Support Console app.

3-Open a Contract Record with record type Client having status as New.

4-Navigate to Summary tab and click on Specify Rate button.

6-Complete the Specify rate flow and submit it.

7-Verify that PCA records should be created and displayed on Summary tab.

Issue 1   If user select  Type of Rate as - Stipend, Variable, Target, then upon filling all the details and submitting the flow, the records are not getting created on the Summary tab.

Expected Result - The records should be created upon successful completion of the Specify Rates flow.

Actual Result - The records are not getting created on the Summary tab.

Issue 2 - On the Contract Record with Record type Client the status fields value are missing.

Expected Result  The Status field should contain correct values as per the AC which are  Status = Submitted, In Progress, Executed, Abandoned, New.

Actual Result - As of now different values are coming under the Status field on contract record having record type Client.
