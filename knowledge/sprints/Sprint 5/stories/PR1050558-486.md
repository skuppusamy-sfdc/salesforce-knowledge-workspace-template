## Description

As a SFO (Staffing Fulfillment Optimization) user, I want the ability to generate facility rates by designating the type of rate and specified rate for each Type & Facility Title combinations, so that we have clearly defined rates for each Service Account.

## Acceptance Criteria

Given I am a SFO in the Corporate Support or Corporate Support Console app

When I navigate to a **Contract** record (record type = **Client**)

And clicked the **Specify Rates** button (part of [https://salesforce.atlassian.net/browse/PR1050558703|https://salesforce.atlassian.net/browse/PR1050558703|smart-link] )

Then I am presented with a Rate Type screen to capture:

- Type of Rate (Contract Payment Agreement, field = Type)

** Required to select one from the following list of values to proceed to the next screen:

*** Stipend

**** **Shift*

*** Variable

*** Target

*** Leadership Incentives

*** Bonus

- Facility Selection

** Select one option between (required):

*** Employed NP/PA

*** Other Facility Titles

Then when I select **Shift**, I see the Rate Details screen and I have the following option to:

- Go back to the previous screen (button = Previous) 

** And if I selected go back to the previous screen (button = Previous), then user is returned to the Type of Rate screen with the previously selected value

- Populate the following to proceed:

*** **Type*

**** **select Type from: (Shift object, filter by Contract record where we are entering rates)* 

**** **Users need to see: Shift Full Name, Shift Day of the Week, Shift Start Time, Shift End Time*

**** **Required, Multi-Select; user shall have the ability to select multiple values.*

** Facility Title (Code Set; Code Set Type = Facility Title)

*** System to display all Facility Title where Active = true

**** When Employed NP/PA is selected, display records where EE NP/PA Rates? = true

**** When Other Facility Titles is selected, display records where EE NP/PA Rates? = false

*** Required, Multi-Select; user shall have the ability to select multiple values.

*** **Time Type (only when Employed NP/PA is selected):*

**** **Field is displayed and required picklist (see data dictionary for values) when selected Facility Title record has the following: EE NP/PA Rates? = true* 

**** Facility Title: _EE Nurse Practitioner, EE Physician Assistant, EE Lead Midlevel Provider_

** Rate

*** Required, editable Currency field (16,0); user to capture whole number

**** Unit of Measure (**required*); per Metric, % Salary Reduction, per Monthly Tier, per Hour, per Shift, per Month

Upon completion (finish) information captured by the user are saved to the Preventive Care Agreement record(s).

## Solution

- Dependent upon [https://salesforce.atlassian.net/browse/PR1050558697|https://salesforce.atlassian.net/browse/PR1050558697|smart-link] 

Create the following records related to the Contract Payment Agreement record; create one per Facility Title & Type captured by the user:

- Preventive Care Agreement

** Facility Title = <Facility Title selected by the user> 

** Shift = <Shift selected by the user>

** Rate = <Rate inputted by the user>

** Time Type = <inputted by the user (note only for type of rate = shift)>

** Unit of Measure = <Unit of Measure selected by the user>

** Bonus Measure Type = NA

** Target Measure Percent = 0

** Bonus Payout = 0
