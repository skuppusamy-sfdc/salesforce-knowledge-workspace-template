## Description

As a Clinician, I need to complete an intake form, so that I can provide necessary information to move forward in the recruiting process and it is retained for downstream onboarding processes

## Acceptance Criteria

Given I am a Recruiter in the Corporate Support or Corporate Support Console app

When I update the ATS Stage field on an ATS Applicant record to Awaiting Intake Form

And the Clinician does not have a Case record (RT = Clinician Intake Form) where **Status = Not Started, In Progress** OR **Status = Closed** and **Completed Date is within last 180 days**

Then the intake form (case) is sent as defined by the Solution.

Given I am a Recruiter in the Corporate Support or Corporate Support Console app

When I navigate to a clinician record

Then I am able to click the Intake Form button

** If the clinician has an intake form where **Completed Date is within last 180 days* display the following message “An intake form was completed on [Completion Date]. Do you want to send another intake form?”

- If none, send the intake form (case) with no prompt

## Solution

**When I update the ATS Stage field on an ATS Applicant record to Awaiting Intake Form**

Use a Record Triggered Flow for Update on Object ATS Applicant,  Field = ATS Stage, Value = Awaiting Intake Form

** Search for Case Record with RT = **Clinician Intake Form* with criteria & ContactId for Clinician

** **Status = Not Started, In Progress** OR **Status = Closed** and **Completed Date >= Today - 180 Days*

** When Found Stop

** When Not Found

*** Create Case record

***#  Subject = Intake Form YYYYMMDD

***# Contact Name = bpats__ATS_Applicant__c.bpats__ATS_Applicant__c <Clinician (ContactID)>

***# Type = Intake Form

***# Status = Not Started

***# Closed Reason = Blank

***#  Account Name = bpats__ATS_Applicant__c.bpats__ATS_Applicant__r.AccountId

***#  Case Owner = bpats__ATS_Applicant__c.bpats__ATS_Applicant__c.OwnerId

***#  Case Origin = automatic for automated creation, manual (default) when manual

***#  Priority = medium

***# Date/Time Opened = Today

----

**When I navigate to a clinician record**

**Then I am able to click the Intake Form button**

Use a ScreenFlow with Button on Person Account (Clinician) name Intake Form 

** Search for Case Record with RT = **Clinician Intake Form* with criteria & Contact ID for Clinician 

** **Status = Not Started, In Progress** OR **Status = Closed** and **Completed Date >= Today - 180 Days* 

*** Pick only the latest record sorted on **Completed Date Desc*

** When Found Stop

*** An intake form was completed on [Completion Date]. Do you want to send another intake form?

*** When user say Yes

**** Create Case record

****#  Subject = Intake Form YYYYMMDD

****# Contact Name = bpats__ATS_Applicant__c.bpats__ATS_Applicant__c <Clinician (ContactID)>

****# Type = Intake Form

****# Status = Not Started

****# Closed Reason = Blank

****#  Account Name = bpats__ATS_Applicant__c.bpats__ATS_Applicant__r.AccountId

****#  Case Owner = bpats__ATS_Applicant__c.bpats__ATS_Applicant__c.OwnerId

****#  Case Origin = automatic for automated creation, manual (default) when manual

****#  Priority = medium

****# Date/Time Opened = Today

*** When user say No

**** Return back to Person Account (Clinician)

Assessment lookup in the Case will be populated in future story.
