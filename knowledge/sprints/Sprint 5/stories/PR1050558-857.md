## Description

As an Enrollment Specialist, I need to access to a Enrollment Specialist queue for cases, so that cases can be associated to a queue.

## Acceptance Criteria

_Story for queue to support cases._

Given that I am a Enrollment Specialist in the Corporate Support or Corporate Support Console App

When I navigate to the Case tab 

Then I can view Enrollment Specialist Queue (shows records that have been assigned to the Enrollment Specialist Queue; not yet assigned to an individual)

- Then I can see the following columns: <Case Number, Subject, Status, Priority, Date/Time Opened>

- Then I can select the case that I want to assign to myself

- Then I can click Change Owner

- Then I can select the new owner

----

This story includes populating the Case Owner with the Enrollment Specialist queue as defined in [Clinician Onboarding  Cases and Tasks|https://docs.google.com/spreadsheets/d/1YC9vi1hRUZfYPY3doZwSIxczV2HtMeLl6jcUrI-qKY/edit?usp=sharing], when we auto generate the Facility Child onboarding cases when the user initiates early onboarding.
