## Description

It appears that my permissions in the Unified Platform have changed as I am no longer able to edit the Bootcamp Enrollment or Completion Date fields. Please update this so I can edit these fields going forward:

Reported by Georgette Walters

## Acceptance Criteria

As a Data Quality Users I can Edit Bootcamp Enrollment and Bootcamp completion dates manually on training records.

## Solution

Add Training permissions to “Manage Clinicians” Permission Set (This Perm Set is already part of the DataQuality Perm Set Group.

**Object** Training__c

**Object Settings**

Read=True

Edit=True

**Read/Edit on Fields:**

- Bootcamp Completion Date	Bootcamp_Completion_Date__c		

- Bootcamp Enrollment Date	Bootcamp_Enrollment_Date__c
