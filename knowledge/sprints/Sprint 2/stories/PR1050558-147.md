## Description

As a Medical Staff Liaison - CPA, I need to manage supervisory relationships, CPA contracts, and legal requirements, so that I can accurately create Collaborative Practice Agreements

## Acceptance Criteria

Given that I am a Medical Staff Liaison - CPA in the Corporate Support app

When I navigate to the Collaborative Practice Agreement tab

Then I am able to click on a **Collaborative Practice Agreement record** and see the following:

Header: Collaborative Practice Agreement Name, Relationship Status

Center Column: tabs are Detail (Default), Contracts

Detail Tab: page layout as specified in data dictionary

Contracts Tab: Contracts related list

Given that I am a Medical Staff Liaison - CPA in the Corporate Support app

When I navigate to the Compliance Rule Assignments tab

Then I am able to click on a **Compliance Rule Assignment record** and see the following: 

Header: Compliance Rule Assignment Name

Center Column: tabs are Detail (Default)

Detail Tab: page layout as specified in data dictionary

Given that I am a Medical Staff Liaison - CPA in the Corporate Support app

When I navigate to the Compliance Rule Assignments tab and create a new record

Then I am able to select between the following **Compliance Rule Assignment record types and see their descriptions**:

+Agreements Required+: Apply this rule if a collaborative practice agreement (or equivalent) is required to practice.

+Practice-Wide Relationship Required+: Apply this rule if all nurse practitioners and/or physician assistants need to be linked to all physicians at the practicing location.

+Supervision Set Point+: Apply this rule if there is a limit to the number of nurse practitioners and/or physician assistants a physician can supervise/collaborate with.

Given that I am a Medical Staff Liaison - CPA in the Corporate Support app

When I navigate to the Contracts tab and the Collaborative Practice Agreement list view

Then I am able to see **Contracts where Record Type Collaborative Practice Agreement**

And I can click on a record and see the following:

Header: Contract Number

Center Column: tabs are Detail (Default), Related

Detail Tab: page layout as specified in data dictionary

Related Tab: text component “placeholder for Conga related list”

Given that I am a Medical Staff Liaison - CPA in the Corporate Support app

When I navigate to a Collaborative Practice Agreement record

Then I am able to view fields that display from Healthcare Practitioner Facility records (Current Facility Title, First Scheduled Date)

## Solution

Create 2 custom objects - LRP and page layout for both objects

Create 3 record types on Compliance Rule Assignment with descriptions (same LRP with different page layouts)

Create Collaborative Practice Agreement record type on Contract
