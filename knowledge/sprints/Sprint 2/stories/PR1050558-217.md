## Description

As a System Administrator, I need to have the same view of records in salesforce as business users, so that I can manage data and troubleshoot issues as needed

## Acceptance Criteria

Given that I am a System Administrator on the Corporate Support or Corporate Support Console apps

When I navigate to the following records:

- Account (Facility/Hospital, Business Account, Competitor, Service Account, Corporate Entity)

- Person Account (Clinician)

- Healthcare Provider NPI, Business License, Healthcare Provider Specialty, Healthcare Provider Taxonomy, Board Certification, Person Education, Person Employment

- Contract (Clinician, Client, Collaborative Practice Agreement)

- Contract Payment Agreement

- Standard Fee Agreement

- Opportunity

- Service

- Service Line

Then I am able to see the same lightning record page and page layout as users with the Corporate Support profile

## Solution

Update LRP assignment for existing pages to add access for System Administrator profile
