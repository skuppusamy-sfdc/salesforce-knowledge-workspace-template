## Description

As a Clinician Contracting Specialist, I need to see Account field on Contract Payment Agreement related lists, so I can easily troubleshoot

## Acceptance Criteria

Given I am a Clinician Contracting Specialist in the Corporate Support apps

When I open a Clinician Contract record and navigate to the **Contract Payment Agreement** related list

Then I am able to see the following columns: Name, Account Name (HPF’s Service Account via PNC), Provider Network Contract, Healthcare Practitioner Facility, Start Date, End Date, Status, Agreement Type

And on a Contract Payment Agreement record, under field = “Provider Network Contract” I can see/view (side by side):

- Account Name (formula)

- Healthcare Practitioner Facility (formula)

And at the bottom of the Contract Payment Agreement record, I can see/view the “System Information” section with the following fields:

- Created By

- Last Modified By

Given I am a Clinician Contracting Specialist in the Corporate Support apps

When I open a Clinician Contract record and navigate to a **Provider Network Contract** record 

Then in the “System Information” section I can see/view the following fields:

- Created By

- Last Modified By

## Solution

Clinician Contract

Create new URL formula field in CPA to show the Account (Facility Name) from PNC>HPF 

Create new URL formula field in CPA to show the HPF Name from PNC>HPF 

Add the Contract CPA Related List View as per AC

Use formula field to create below as the system don't have these field out of the box

- Created By

- Last Modified By
