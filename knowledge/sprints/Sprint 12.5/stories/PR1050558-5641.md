## Solution

- 4 new Fields in HPF 

** `Is_Sister_Site_Location__c  Checkbox`  is true when sister-site links exist for that HPF account

** `Inactivity_Qualified_270__c  Checkbox`   is true when crosses over 270 days threshold 

** `Re_Contract__c - Checkbox`

** `Aging_Days_Left__c - Number `

*** calculate inactivity days for current HPF non Sister Location

*** calculate minimum inactivity days across sister sites 

**** choose the **minimum of those two* as the merged inactivity duration set `Aging_Days_Left__c = max(0, 365 - mergedDuration)`

Logic:

Create a Apex batch job

Sope:

Query all ContactId from HPF where Status NOT IN ('Prospect', 'Candidate', 'Superseded', 'Inactive') AND EffectiveTo == NULL  

Execute Scope:

1. Query all HPF for the Scope

1. Loop through HPF

## Collect Set<Id> → [HPFId]

## Collect Map<PractitionerId, HPF> → [HPF]

## Collect Map<PractitionerId, Set<AccountId> → [PractitionerAccount]

## Collect Map<AccountId> → [ServiceAccountId]

1. Utilize the SisterSiteGroupingUtility to get SisterSiteGroup Map<AccountId, String> [accountIdToName];

1. Query Contract record for using [HPFId] Query: Select Contract.AccountId, Contract.Employment_Type__c, Min(Contract.Initial_Effective_Date__c) From PNC Where  Healthcare_Practitioner_Facility__c = [HPFId] Group by Contract.AccountId, Contract.Employment_Type__c

## Loop through and collect Map<ClinicianId, Map< Employment_Type__c, Initial_Effective_Date__c>> **[ContractEffectiveDate]**

1. Loop Through the [HPF] in Scope **COLLECTING DATES FOR SISTER SITES**

## If [HPF.|http://HPF.id]Id in [accountIdToName]

## Build a Map<ClinicianId, Map<SisterSiteGroupName, LastWorked> **[ClinicianSisterLastWorked]**

###  Compare if there is already a LastWorked for the SisterSiteGroupName and if Loop.LastWorked >  value in [ClinicianSisterLastWorked] then update [ClinicianSisterLastWorked].LastWorked = Loop.LastWorked

## Build a Map<ClinicianId, Map<SisterSiteGroupName, NextScheduledDate> **[ClinicianSisterNextScheduledDate]**

###  Compare if there is already a NextScheduledDate for the SisterSiteGroupName and if Loop.NextScheduledDate >  value in [ClinicianSisterNextScheduledDate] then update [ClinicianSisterNextScheduledDate].NextScheduledDate = Loop.NextScheduledDate

## Build a Map<ClinicianId, Map<SisterSiteGroupName, HospitalPrivilegedDate> **[ClinicianSisterHospitalPrivilegedDate]**

### Compare if there is already a HospitalPrivilegedDate for the SisterSiteGroupName and if Loop.HospitalPrivilegedDate >  value in [ClinicianSisterHospitalPrivilegedDate] then update [ClinicianSisterHospitalPrivilegedDate].HospitalPrivilegedDate = Loop.HospitalPrivilegedDate

1. Loop Through the [HPF] in Scope

## If HPF is Part of Sister Site 

### Compare using [ClinicianSisterLastWorked], [ClinicianSisterNextScheduledDate],  [ContractEffectiveDate], [ClinicianSisterHospitalPrivilegedDate]

### Based on comparison result stamp  Is_Sister_Site_Location__c, Aging_Days_Left__c for each HPF

### If the Date crossed 270 then set 

#### Inactivity_Qualified_270__c = Y

#### Re_Contract__c = Y

### If Date Crossed 365 then 

#### Status = Inactive

#### Credentialing Status = Inactive

#### Contracting Status = ‘Terminated’

#### Termination Date = <day the automation ran>

#### Inactive Reason = ‘Provider Inactivity >12 Months'

#### Effective To = <set to 365 days from the 'Last Date Worked'>

####* If HPF’s ‘Last Date Worked’ is blank, then set to 365 days from the ‘Initial Effective Date’ (Clinician Contract)

####* If Clinician Contract’s ‘Initial Effective Date’ is blank, then set to 365 days from the HPF’s 'Hospital Privileged Date'
