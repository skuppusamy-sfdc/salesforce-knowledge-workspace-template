## Description

Note, A View All Reports Permission Set already exists. These two Permission Sets are in addition

**Creations of two Reporting Permission Sets**

1. **Name: Create and Manage Personal Reports**

Create and Customize Reports = TRUE

Create Report Folders = TRUE

Export Reports = TRUE

Edit My Reports = TRUE

Report Builder (Lightning Experience) = TRUE

1. **Name: Create and Manage Public Reports**

Manage Reports in Public Folders = TRUE

Schedule Reports = TRUE

Subscribe to Reports = TRUE

Subscribe to Reports: Add Recipients = TRUE

Subscribe to Reports: Send to Groups and Roles = TRUE

Subscribe to Reports: Set Running User = TRUE

1. **Update Perm Set Group: Health Cloud Restricted User Group**

Add both Perm Set Groups “Create and Manage Public Reports” and “Create and Manage Persona Reports” to the “Health Cloud Restricted User Group” Permission Set Group

1. **Update Perm Set Group: Corporate Support**

Add both Perm Set Groups “Create and Manage Public Reports” and “Create and Manage Persona Reports” to the “Corporate Support” Permission Set Group

## Acceptance Criteria

As a system administrator….

AC1 A Permission Set Named “Create and Manage Personal Reports” is available to be assigned to users and the listed system permission are set to true

AC2 A Permission Set Named “Create and Manage Public Reports” is available to be assigned to users and the listed system permission are set to true

AC3 Both Permissions Sets named above are visible in the “Permission Sets In This Group” section of the “Health Cloud Restricted User Group” Permission Set Group

AC4 Both Permissions Sets named above are visible in the “Permission Sets In This Group” section of the “Corporate Support” Permission Set Group
