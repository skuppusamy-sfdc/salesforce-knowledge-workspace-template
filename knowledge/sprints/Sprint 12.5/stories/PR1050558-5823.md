## Description

As a System Administrator I want to add new HPF section = ‘Insurance Information' in layout, so that Data Quality team can edit the fields within the 'Insurance Information’ section.

## Acceptance Criteria

AC1: Update HPF layout - Add Section = “Insurance Information”

Given I am a system administrator within the Corporate Support apps

When an internal Corporate Support user with access to HPF (Healthcare Practitioner Facility) **views** a record

Then they will see a new section = “Insurance Information” above “Additional QGenda Data”

And within the new section = “Insurance Information”, I can see the following fields:

- Insurance Provided By

- Certificate of Insurance Start

- Certificate of Insurance Expiration

And the fields listed above are NO longer listed in the “Additional QGenda Data” section.

AC2: **Edit** access on section = “Insurance Information” for ONLY users with Permission Set Group = 'Data Quality' and ‘Risk Management Group’

Given I am a user in the Corporate Support app

And I have Permission Set Group = 'Data Quality' OR ‘Risk Management Group’ 

When I am on a HPF record

Then I can edit the following fields within section = “Insurance Information”:

- Insurance Provided By

- Certificate of Insurance Start

- Certificate of Insurance Expiration

NOTE for AC2: if the user does not have the Permission Set Group = 'Data Quality' and ‘Risk Management Group’, then they should NOT be able to edit the fields listed in AC2
