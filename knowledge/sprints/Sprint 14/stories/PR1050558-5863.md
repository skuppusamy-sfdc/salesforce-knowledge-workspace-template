## Solution

**SOLUTION**

1. Update `HPF_Updates_Triggered_Flow` -- "Is Inactive" branch

After Assign_PE_instance_for_Inactive_HPF, add:

a) Decision: `Is_TerminationDate_Populated`

• Condition: $Record.TerminationDate IsNull = false

• Yes → proceed to Get Account Team

• No → skip to Assign_InactiveDate_PE_to_HPF_Updates_PE_List (existing)

  b) Get Records: `Get_Account_Team_Members`

  • Object: AccountTeamMember

  • Filter: AccountId = $Record.AccountId AND TeamMemberRole IN (Medical Staff Coordinator, Clinician Experience Specialist, Medical Staff Liaison  CPA, RecruiterPrimary) AND User.Email IsNull = false

  c) Decision: `Has_Account_Team_Members`

  • Condition: Get_Account_Team_Members IsNull = false

  • Yes → proceed to Loop

  • No → skip to Assign_InactiveDate_PE_to_HPF_Updates_PE_List (no recipients, no point sending)

  d) Loop + Assignment: Collect emails

  • Loop over Get_Account_Team_Members

  • Assign CurrentItem.User.Email → add to var_TerminationEmailRecipients (text collection)

  e) Get Records: `Get_Inactivations_Email_Config`

  • Object: Email_Address_Configuration__mdt

  • Filter: Label = 'inactivations' AND IsActive__c = true

  • (CC is optional -- if null, email still sends without CC)

  f) Get Records: `Get_OrgWide_NoReply`

  • Object: OrgWideEmailAddress

  • Filter: DisplayName = 'No Reply'

  • (Already queried elsewhere in the flow - reuse variable if in scope, otherwise requery)

  g) Decision: `Has_OrgWide_Email`

  • Condition: Get_OrgWide_NoReply IsNull = false

  • Yes → proceed to send email

  • No → skip to Assign_InactiveDate_PE_to_HPF_Updates_PE_List (can't send without sender)

  h) Action: `Send_Termination_Email` (emailSimple)--> Create Template [Select Template]

  • To: var_TerminationEmailRecipients

  • **CC: Get_Inactivations_Email_Config.Email__c (if not null) AND IF Field “Inactive Reason“ EQUALS “Provider Inactivity > 12 Months“**

  • Subject: see mock in AC

  • Body: see mock in AC

  • Sender: Get_OrgWide_NoReply.Id

  • Fault path → Log_Termination_Email_Error

  i) Action (fault): `Log_Termination_Email_Error`

  • Apex Action: ErrorLogUtil (Create Error Log)

  • Error Message: {!$Flow.FaultMessage}

  • Severity: Medium

  • Source Component: HPF_Updates_Triggered_Flow

  • Connect to → Assign_InactiveDate_PE_to_HPF_Updates_PE_List (continue flow, don't block)
