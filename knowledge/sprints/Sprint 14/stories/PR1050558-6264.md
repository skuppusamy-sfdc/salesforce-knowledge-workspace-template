## Description

Please create two fields on the Salesforce Integration_Log__c object

Error_Code__c   Text(255)

Retry_Count__c  Int

The error code will discretely store the 400, 404, 500, etc error number. May include text in the future, so dont limit to numbers

The Retry count will be incremented via a trigger flow whenever a log is retried. This is to report on and prevent excessive retries done manually or auto retry if implemented in the future
