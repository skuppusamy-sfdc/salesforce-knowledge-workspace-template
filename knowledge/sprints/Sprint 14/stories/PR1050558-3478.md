## Description

As a Clinician Contracting Specialist user, I need to initiate rate change notices for clinicians based on changed facility rates so that I can be guided through the process with respective clinicians quickly and seamlessly.

## Acceptance Criteria

This story is to enhance what was built in story [https://salesforce.atlassian.net/browse/PR10505582084|https://salesforce.atlassian.net/browse/PR10505582084|smart-link] 

Given I am a Clinician Contracting Specialist within the Corporate Support Apps

When I initiate Agentforce for Clinician Rate Change Notices

And select the Service Account & the Practice Operations (PO) contract 

Then Agentforce shall display a list of affected **Facility Title(s)** based on “Practice Operation Changes” record(s) related to the PO Contract that meet the following criteria:

- Process Type = “Agent”

And Agentforce shall display a list of **impacted clinicians** for that selected Facility Title(s), based on related “Client Change Management” records that meet the following criteria:

- Process Type = “Agent”

- Status = “New”

And for each clinician where Agentforce was able to successfully complete ALL steps for a Rate Change Notice, then Agentforce shall update the “Client Change Management” record Status = ‘Completed’. 

Note: 

- We set ‘Process Type’ = “Agent” when: ONLY Rate Increased Existing Shift (All are same rate for each shift type for that facility title)
