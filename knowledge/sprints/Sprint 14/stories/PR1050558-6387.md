## Description

Issue - When converting a Clinician from IC to EE via contract cloning, the system creates an unnecessary Approval Work Item (AWI) for CCE Mismatch.

Expected Behavior:

The system filters out the deleted IC record and the system identifies that the 3 active HPF records have the same CCE thus Contracting Corporate Entity is populated on the Contract.

No AWI is created.

Also, if CCE is not matched then as per AC1 logic AWI should be created by treating it as a fresh identity.

Actual Behavior:

The system triggers a Mismatch. An Approval Work Item is created for the Corporate Entity Mismatch Queue.

Steps to Reproduce:

1-Open an existing IC Contract.

2-Use the guided experience to Clone/Start New Request for Employment Type = "Employee".

3Add 3 new facilities  for emp type  EE with the same CCE (e.g., "CCE1") and delete the IC facility.

4-Now Click Submit Contract Request.
