## Description

**Current State:**

- After a clinician completes all tasks in Experience Cloud, intake status remains as "Not Started" or "In Progress."

- Recruiters must manually update the status to "Completed," which frequently does not happen in a timely manner.

- No automated notification is sent to the recruiter when intake is complete.

**Business Impact:**

- Inaccurate pipeline visibility: recruiters don't know when clinicians are ready to advance. Manual updates create data quality issues downstream in reporting and scheduling.

**Requirement:**

- When all clinician-assigned tasks in Experience Cloud are completed, automatically update the intake/case status to "Completed."

- Trigger a recruiter notification (in-app and/or email) when the status moves to Completed.

**Suggested Acceptance Criteria:**

-◦     Given a clinician completes all assigned intake tasks, when the last task is marked complete, then the intake status updates to Completed without recruiter action.-

-◦     Given the status updates to Completed, a notification is sent to the assigned recruiter within a defined SLA (e.g., 15 minutes).-

-◦     Status update is visible on the Kanban board and Clinician record.-

-◦     If a task is uncompleted after prior completion, status reverts accordingly (edge case to validate).-

## Acceptance Criteria

As a recruiter my intake case is automatically set to complete and I receive a notification when a clinician has saved their final intake assessment

## Solution

**Record Triggered Flow:** Assessment   After Insert (Update this Flow)  Write the logic in Asynch Path 

Any time an AssessmentEnvelopeItem is updated to status = Completed, query for any other AssessmentEnvelopesItem and when each of them are completed then update Get the Case Id from AssessmentEnvelope and update the Case as Closed

If all AssessmentEnvelopes are Compelted, update the Case as “Closed” 

AND

Create a Task with the following details

Subject = Intake Case Closed For Case.Account.FullName

Assigned to = Case.Owner

Due Date = TODAY
