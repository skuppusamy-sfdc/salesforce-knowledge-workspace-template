## Description

Issue - The system is incorrectly creating an Approval Work Item (AWI) for the "Corporate Entity Mismatch Queue" even when all related HPF records share the same Contracting Corporate Entity (CCE). The AWI should only be triggered if a mismatch exists.

Expected Behavior

The system evaluates all related HPF records. Because all CCEs match, the Contract’s Contracting Corporate Entity field should be automatically populated with that entity.

No Approval Work Item should be created for the Corporate Entity Mismatch Queue.

Actual Behavior

The system incorrectly identifies a mismatch.An Approval Work Item (AWI) is generated for the Corporate Entity Mismatch Queue.

Steps to Reproduce:

1-Initiate a Contract Request for a Clinician.

2-Navigate to the Contract Selection step and select Start New Request.

3-Add multiple Facilities (HPF records) to the request that all belong to the same Contracting Corporate Entity (e.g., all facilities belong to "CCE A").

4-Ensure PNC Delete is set to False/Unchecked for all these records.

5-Click Submit Contract Request after adding the Configure Rates.
