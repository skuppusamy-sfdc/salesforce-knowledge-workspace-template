## Description

Steps to reproduce:

1. Login to Core HC Org as Legal - Atticus Finch

1. Open a service account with Clinician contracts

1. Navigate to Related tab and verify the ‘Healthcare Practitioner Facilities’ related list

Expected Result:

All personas should be able to view the Time Type column on HPF related list

Actual Result:

Below personas do not have access to view the Time Type column on HPF related list

1. Legal

1. Field Ops/SFO

1. Business Development Associate

## Solution

Update the **Corporate Support** permission set to grant **Read access** to the **All Fields** on the **Healthcare Practitioner Facilities** object, then re-deploy and validate for all personas.

## Root Cause

Read access for the **Time Type** field on the **Healthcare Practitioner Facilities** object is missing in the **Corporate Support** permission set. This permission set is part of the permission set group assigned to all personas, so users do not have visibility to this field.
