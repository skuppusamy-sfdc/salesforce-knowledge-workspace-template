## Description

On an Opportunity (RT:Growth Opportunity), when the Sales VP selects “Initiate Contracting Request” guided experience and selects contract type = Letter of Intent

Expected Result: 

1. 'Name of each Hospital' should be a table displaying all active opportunity services

1. ‘Service Line' is a read only display of service lines (distinct values) separated by commas, from the active Services related to the opportunity)

## Root Cause

The required component version containing the logic fix for displaying service lines correctly was not successfully migrated to the UAT2 environment. This version mismatch resulted in the application running outdated or incorrect code, leading to the display bug for the “Letter of Intent” contract type.
