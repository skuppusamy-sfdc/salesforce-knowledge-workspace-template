## Description

As a system administrator, I want to ensure that users with the ‘View Client Onboarding’ permission set have access to the PMT Project record, but should NOT have access to the Opportunity record. Also, Implementation Specialist shall only have edit access to the Opportunity field = “PMT Project”, so that we ensure they are not inadvertently making other updates.
