## Description

As a Medical Staff Liaison, I need task records to be closed automatically, so that I do not have to do duplicate work in multiple systems

## Acceptance Criteria

Given I am a MSL in the Corporate Support apps

When a Clinician onboarding is initiated

And the “I9 Update” task is created for the ‘HR Setup’ child case (RT: Clinician Onboarding)

And the “I-9 Status” field is updated to “Complete” on the related Person Account (Clinician) record

Then the “I9 Update“ task record shall update to Status = “Completed”

NOTE: for Permission Set = Manage Clinicians, even if the Permission Set is giving Read, Create & Edit on the object, we are just giving Read access on the Field, since this field is to be edited by integration.
