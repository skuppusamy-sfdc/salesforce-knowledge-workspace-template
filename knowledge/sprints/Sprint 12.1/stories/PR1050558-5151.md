## Description

As an Integration User, I need facility title changes to automatically create facility offboarding cases, so that downstream systems are updated with the latest clinician information

## Acceptance Criteria

Given I am a Recruiter

When a clinician contract has a HPF included where the clinician has an active HPF (Status ≠ ‘Inactive’) for that same Service Account (ex. Facility Title change)

Then when the Contract is Executed the following case record is created automatically:

- Subject = Facility Offboarding  <Clinician Name>  <Service Account Name>

- Contact Name = Clinician

- Type = Facility

- Status = Closed

- Close Reason = Complete

- Rehirable = TRUE

- Account Name = <Service Account from original HPF>

- Healthcare Practitioner Facility = <original HPF>

- Date Termination Received = <Contract record created date>

- Termination Request Date = <Contract record created date>

- Last Day = <new contract start date - 1>

- Reason for Termination = Contract Termination

- Voluntary Termination = TRUE
