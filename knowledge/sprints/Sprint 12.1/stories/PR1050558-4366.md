## Description

As a system administrator, I want to change Healthcare Practitioner Facility field = "Received Current Completed References" from a Date field to a Checkbox and update flow, so that down stream systems are not affected.

In current state it is a checkbox that goes from QGenda to SF that the MSL uses as a quality check to ensure References are updated.

## Acceptance Criteria

AC1: Change data type from a Date field to a Checkbox

Given I am a System Administrator

When I am working with object = Healthcare Practitioner Facility

Then field = "Received Current Completed References" should be changed from a Date field to a Checkbox

And API shall remain the same “Received_Current_Completed_References__c”

AC2: Update any related flows

Given I am a System Administrator

When I am working with object = Healthcare Practitioner Facility

And Checkbox field = "Received Current Completed References"

Then any related

- Flow(s) are update to support checkbox field.

- Flexipage

## Solution

AC1: Change the data type of the Field 

AC2: Change the logic using the field from ISCHANGED(Date Field) to Received_Current_Completed_References__c = True
