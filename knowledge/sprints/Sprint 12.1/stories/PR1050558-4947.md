## Description

**Summary:**

In the cloned Healthcare Practitioner Facility (HPF) record, the **Variable Rate section** displays duplicate rate values instead of showing the expected strikethrough (old rate) and active (new rate) format.

**Steps to Reproduce:**

1. Open an existing HPF record (e.g., _plkd gov_).

1. Update the **Variable Rate** field (Encounter) from **$70 per Metric** to **$100 per Metric**.

1. Clone the HPF record.

1. Open the cloned HPF and navigate to the **Variable** → **Encounter** sectio

**Expected Result:**

The cloned HPF should display rates in the same format as the original contract:

** New rate (**$70 per Metric**) shown as **active/current* value.

**Actual Result:**

The cloned HPF displays the rate **$70 per Metric** twice — both as strikethrough and as the current value — instead of showing the correct rate transition.
