## Description

Steps to Reproduce:-

1-Login as SFO Persona into the Core HC org and navigate to the Corporate Support apps.

2-Navigate to the PO Contract via Opp record.

3-Open the PO Contract and attempt to update Contract End Date field.

Issue - SFO Persona does not have Edit access on Contract End Date field.

Expected Result - SFO Persona should have Edit access on Contract End Date field.

Actual Result - SFO Persona does not have Edit access on Contract End Date field.

Persona - Field Ops/SFO – Richard Caramel

## Solution

**Note:** **This was not mentioned initially in the user story. Later on added.**

Give the edit access to **Contract.End Date** field via this **Manage Client Contracts** permission set.

## Root Cause

**Note:** **This was not mentioned initially in the user story. Later on added.**

The user did not have edit access to the **Contract.End Date** field in the QA org, which caused the issue.
