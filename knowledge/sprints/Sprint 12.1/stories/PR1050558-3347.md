## Description

As a Legal team member, when I am on a Client Contract, I want the ability to Expire, so that I can indicate that a contract is expiring and a NEW agreement/contract is created. (not amending, the contract has ended. A brand new contract is being created or has to be created)

## Acceptance Criteria

AC1: Auto task reminders - Upcoming Contract Expiration 

Given I am a Legal Team Member within the Corporate Support Apps

When a Client Contract has a “Term & Renewals” record where the “**Internal Renewal Start Date**” is equal to the current date/today’s date

Then the system shall **create one task per related Service Account** on the date listed in field = “Internal Renewal Start Date”: (note: the PNC record is the relation between the Client Contract and the specific Service Account) 

- Record Type: Default

- Subject: <Service Account> Upcoming Contract Expiration 

- Related To: Client Contract 

- Assigned To: Client Contracting Queue

- Due Date: Create date + 14 days

- Priority: Normal

## Solution

**AC1 & 2:** 

**Name:** Client Contract Scheduled Flow

Use Scheduled Flow in Contract Object and Filter Conditions for RT: Client Contract AND Contract_Type__c = “Terms & Renewals” AND (Internal Renewal Start Date = Today() OR Renewal Notice Date = Today())

For Each Record that Qualifies Criteria

When Internal Renewal Start Date = Today()

- Query Contract_Provider_Network_Relation__c.Contract__c = CURRENT[.Contract.Id|http://Current.Contract.Id] For each records create Task record(s)

** Record Type: Default

** Subject: Provider_Network_Contract__r.Service_Account__r.Name + Upcoming Contract Expiration

** Related To: Client Contract

** Assigned To: Client Contracting Queue

** Due Date: Create date + 14 days

** Priority: Normal

When Renewal Notice Date = Today()

- Query Contract_Provider_Network_Relation__c.Contract__c = CURRENT[.Contract.Id|http://Current.Contract.Id] For each records create Task record(s)

** Record Type: Default

** Subject: Provider_Network_Contract__r.Service_Account__r.Name + Upcoming Contract Expiration REMINDER

** Related To: Client Contract

** Assigned to: Client Contracting Queue

** Due Date: Create date + 7 days

** Priority: High

AC3:

New Button in Contract RT: Client Contract AND Contract_Type__c = “Master” (as part of this story create the Ultiamte_Parent_Id__c field)

On Button Click 

Query Contract.Ultiamte_Parent_Id__c = CURRENT.Contract.Ultiamte_Parent_Id__c

Update all returned records Contract.Status =  Expired
