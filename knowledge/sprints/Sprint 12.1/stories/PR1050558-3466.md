## Description

As a Legal team member, when I am on a Letter of Intent Contract, I want the ability to Amend, so that I can indicate that a contract was amended and a new contract was clone and modified from it.

## Acceptance Criteria

AC1: Ability to ‘Amend Contract'

Given I am a Legal Team Member within the Corporate Support Apps

When navigate to a Contract (RT: Letter of Intent) record

Then I have the ability to select the ‘Amend Contract’ guided experience

And the user is **required** to input:

- Contract Name

- Description

And available navigation options:

- Cancel (cancel is via the “X” in upper right)

** No records are created or updated

** Guided experience is closed

** User is returned to the Letter of Intent Contract

- Finish/Save

** Guided experience is closed

** User is returned to the NEW Letter of Intent Contract (Design: user is presented with a link to view new contract)

** System clones the Contract

*** Clone everything EXCEPT:

**** Contract Name - Populate with ‘Contract Name’ captured in guided experience

**** Description - Populate with ‘Description’ captured in guided experience

**** Key Dates - All fields within this section should be Blank

**** Status - set to 'Draft'

** Need ability to see original files (all files related to the amended contract - contract that was cloned from)

----

(QA Testing Note: ensure that we retest the “Activate Contract” action once this story gets implemented)

## Solution

Flow Action: Amend Contract

UI Inout Element

- Contract.Name → Contract Name: Field Field

- Contract.Description → - Populate with ‘Description’ captured in guided experience

Default setup

- PreviousContractId = CURRENT.Contract.Id

- Contract.Status = 'Draft'

- Key Dates - All fields within this section should be Blank

** Requested Date 

** Date Sent For Internal Review

** Other Party Sent Date 

** Other Party Returned Date

** Executed Copy Mailed Out Date

** Activated Date 

** Activated By

** Date Distributed

Need ability to see original files (all files related to the amended contract - contract that was cloned from)

- Use the ContentDocument and ContentDocumentLink to relate the field
