## Description

A regression has been identified where the **Configure Rate** flow fails to display correct rates when accessed the flow for a second time. (For e.g. Rate change/Deletion of rates) 

 **Actual Result:** Correct rates only appear on the first attempt; subsequent attempts show incorrect data.

**Expected Result:** Rates should fetch and display accurately every time the flow is initialized.
