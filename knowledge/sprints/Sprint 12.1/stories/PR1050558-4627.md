**Description:**

When configuring an initial Client Contract, a PNC record is created without linking the SCP Corporate Entity from the related Service Account. The user is required to enter this information. The function is automated in the current system eliminating any user error by pulling the related SCP Corporate Entity, which is entered and managed by the Corporate Entity team, from the related Service Account.  

We also need a better understanding of the relationship of Healthcare Provider and Healthcare Payer Network fields.

2. User can drill into a Case

4. If this is an initial Contract request, User is presented with all respective Service Accounts

6. Upon Finish, a new Client Contract is created in "Draft" status
