## Description

As a Medical Staff Liaison, I need to terminate a clinician’s employment at a specific facility in Salesforce, so that I can accurately update the status of employees at facilities that SCP staffs for

## Acceptance Criteria

Given I am a Medical Staff Liaison in the Corporate Support apps

When I view a Clinician record

Then I am able to select Initiate Termination Request and see the following:

- Facility Selection  [https://salesforce.atlassian.net/browse/PR1050558701|https://salesforce.atlassian.net/browse/PR1050558701|smartlink]

- Termination Details  [https://salesforce.atlassian.net/browse/PR1050558306|https://salesforce.atlassian.net/browse/PR1050558306|smartlink] AND [https://salesforce.atlassian.net/browse/PR10505582471|https://salesforce.atlassian.net/browse/PR10505582471|smart-link] 

*** **If I have checked the Backfill Recommended checkbox*

** AND if Time Type on HPF is Full Time OR Part Time

- Then when I save, the selected job record(s) are cloned with the same values except the following updates:

** Status = Draft

** Post Externally = FALSE

** Posted Date = null

** If Voluntary = TRUE, Vacancy Reason = Attrition - Resignation 

** If Voluntary = FALSE, Vacancy Reason = Attrition - Termination

- Send an Email to the VPO with the following: 

** Subject: Clinician Termination - Review Draft Job

** Body:

A clinician has been terminated and a draft job record has been created. Please review and update the newly created job record for recruiters to begin the recruiting process. [include link to newly created job record]

_update create HPF from Job automation_ [https://salesforce.atlassian.net/browse/PR1050558823|https://salesforce.atlassian.net/browse/PR1050558823|smartlink]  _to populate Job lookup on HPF for reference  read only Lookup field to job object_

## Solution

-Case RT: Clinician Offboarding-

-When Case.Backfill_Recommended__c is updated and set as True- 

-then **CLONE** **ALL Field** the bpats__Job__c record identified by Healthcare_Practitioner_Facility__r.Job_Applicant__r.bpats__Job__c-

-Set fields- 

- bpats__Job__c.bpats__Status__c = Draft

- bpats__Job__c.bpats__Post_Externally__c = FALSE

- bpats__Job__c.bpats__Posted_Date__c = null

- If Case.Voluntary_Termination__c = TRUE, bpats__Job__c.Vacancy_Reason__c = Attrition  Resignation- 

- If Case.Voluntary_Termination__c = FALSE, bpats__Job__c.Vacancy_Reason__c = Attrition  Termination-

-SendEmail to VPO as per AC-

See Solution in [https://salesforce.atlassian.net/browse/PR10505584143|https://salesforce.atlassian.net/browse/PR10505584143|smart-link]
