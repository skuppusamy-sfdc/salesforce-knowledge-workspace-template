## Root Cause

Determined to not be a MuleSoft error.  

Integration Logic Failure (Mapping Error): The integration process correctly retrieves the Entity EIN (e.g., "27-1578338") from Salesforce. However, the logic responsible for matching this Entity EIN to the corresponding Workday Company Reference ID (e.g., "NW_Florida_Emerg_Phys_LLC") failed or was misconfigured.

Schema Mismatch/Data Retrieval: The core issue is that the system relies on the Entity EIN as the key to look up the correct company. When the wrong Company Reference ID is retrieved (due to a failure in the mapping table, logic, or lookup process), Workday incorrectly associates the clinician with the wrong company, leading to the display error.
