## Solution

**THis field though its a Text its going to have numbers so when testing please use NUMBERS and no alfa numeric or plain Text**

Background: SCP is running into issues when trying to covert Global_Contact_ID__c, Global_Account_ID__c from Number to Auto Number. 

Covert the field data type from 

Contact.Global_Provider_ID__c: Number -> Text

Contact.Global_Contact_ID__c: Number -> Text

Account.Global_Facility_ID__c: Number -> Text

Account.Global_Account_ID__c: Number -> Text

**Any code in the system should only refer to Global_Provider_ID__c or Global_Facility_ID__c** 

1. **No code should refer to fields Global_Contact_ID__c or Global_Account_ID__c**

## **if there a code refer Global_Contact_ID__c then it should be re pointed to Global_Provider_ID__c**

## **if there a code refer Global_Account_ID__c then it should be re pointed to Global_Facility_ID__c**
