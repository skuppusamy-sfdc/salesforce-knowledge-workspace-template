## Description

Steps to Reproduce:

1-Login as HCRU and navigate to Clinician record.

2-Navigate to highlight panel and start the Initiate Early Onboarding flow.

Issue - When logged in as a HealthcloudRestrictedUser, the Initiate Early Onboarding flow begins to load but then closes automatically without completing.

Expected Result - The flow should be displayed and user should be able to complete it.

Actual Result - Flow is getting close as soon as the user starts it.

## Root Cause

ClinicianGetManualOnboardingDetailsIP was using the reusable IPs to fetch the clinician details, and that IP querying the other object **BoardCertification and BusinessLicense** which **Restricted user doesn’t have access to.**
