## Description

**Tester:** Nikhil Kokate

**Persona:** Recruiter

**Test Case ID:** INT11A

**Description:** During SIT testing, when the facility title is set to “Ambassador Physician”, no corresponding Workday integration logs are generated.

**Steps to Reproduce:**

1.Navigate to clinician record 

2.Validate HPF is associated with Valid details 

3.Create contract in Draft state 

4.Add payrates and Submit contract request and Contract will be moved to approved 

5.Navigate to associated contract and create conga agreement 

6.Complete the Docusign approval process and Contract will be marked as executed

**Expected vs Actual Result:**

Expected result 

•The integrated data should be visible in Datadog logs

Actual result 

•Data is not orchestrated to the workday 

**Environment:** UAT

## Root Cause

Integration Mapping Update

 The facility title "Ambassador Physician" was not included in the predefined mapping rules used by the integration service (Mulesoft/process layer) to determine the corresponding Supervisory Organization ID for Workday.

## Root Cause Type

Deployment and Configuration: Incorrect Configuration
