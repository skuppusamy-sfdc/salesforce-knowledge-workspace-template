## Description

**Tester:** Nicole Fontenot

**Persona:** Legal - Client Contracting

**Detailed Description:** In the current system, all SUNs, Facility Terminations, Program Conversions, Name Changes, Address/Ownership Changes issued by Legal are uploaded to the related Service Account. The future system does not allow for any type of document upload other than a Client Contract or Letter of Intent. We need the ability to upload these related documents to the Service Account and/or PNC. This feature provides any user a timeline and supporting documentation of changes to a Service Account. Similar to the need for a copy of a signed Client Contract, the organization needs access to documentation of program changes in the event of an audit, claim or litigation. If this feature is not available at go live, who will own the responsibility of managing this documentation until the enhancement is available?
