## Description

As a Recruiter, I need HPFs meeting certain criteria to be automatically included in clinician contracts, so that they are hired appropriately

## Acceptance Criteria

Given I am a Recruiter in the Corporate Support apps

When I **request a clinician contract** for a clinician with a Contract in Executed Status for the selected Employment Type

Then I am unable to select Start New Contract and I **must Start From Previously Active Contract**

And if an HPF exists where Status = Candidate, Active, Waiting To Be Scheduled, WTBS Here-WRKG at Sister Site, which is not included in the Contract, I am presented with the following warning “This clinician has missing HPF(s) which is not included in the Contract. Please review the Facilities tab on the clinician person account and add missing HPF(s).”
