## Description

As an Integration User, I need to know when a IC clinician is leaving SCP, so that I can remove that clinician from Workday 120 days from date the SCP Clinician Offboarding case is in Status = Closed & Closed Reason = Complete.

## Acceptance Criteria

Given I am an Integration User in the HC org

When we have a case record meeting the following criteria:

- Case Record Type = ‘Clinician Offboarding’

- Type = ‘SCP’

- Employment Type = ‘Independent Contractor’ (IC) 

AND the Case meeting the above criteria is in Status = Closed and Closed Reason = Complete

Then 120 days from the date the criteria is met, a Platform Event = ‘Clinician Offboarding Request’ and Type = “Independent Contractor” shall be created (existing platform event [+PR10505582462+|https://salesforce.atlassian.net/browse/PR10505582462] & [+PR10505584347+|https://salesforce.atlassian.net/browse/PR10505584347])

## Solution

Remove the path of Flow  that sends the Platform Event for IC built as part of [https://salesforce.atlassian.net/browse/PR10505582462|https://salesforce.atlassian.net/browse/PR10505582462|smart-link] 

Create a Scheduled Triggered Flow that triggered 120 Days Delay from the Date when Case.ClosedDate

Send the Platform Event as specific in AC 

Create a Platform Event

-  Name: Clinician Offboarding Request (Use same API name in Story - Integration Dependency)

** API Name: Clinician_Offboarding_Request__c

Use the Contract Employment Type in Case to determine if IC or EE use case

1. Case > Contract > Employee Type = “Independent Contractor” Then

## SELECT Corporate_Tax_ID__c, Corporation_Name__c, Type__c , Contact.FirstName, Contact.MiddleName, Contact.LastName, Contact.Professional_Title__c FROM Corporation_and_Tax_Information__c WHERE Clinician__c = Case.ContactId AND Expiration_Date__c <> null AND Status__c = “Executed' ORDR BY Effective_Date__c DESC LIMIT 1 

## Create a Platform Event record Clinician_Offboarding_Request__e

### Type = “Independent Contractor”

### Business Entity Name = if(Type__c = “Individual” Then  Contact.FirstName + “ “ + Contact.MiddleName + “ “ + Contact.LastName + “ “ + Contact.Professional_Title__c  Else Corporation_Name__c)

### Global Provider Id = Case.Contact.Global_Provider_ID__c

### Tax Id = If Corporation_and_Tax_Information__c .Type__c = "Individual" then Clinician_r.SSN_Encrypted__c & If Corporation_and_Tax_Information__c .Type__c = "Business Entity" then Corporation_and_Tax_Information__c.Corporate_Tax_ID__c

### Reason Description = Case.Description

Set the Delayed Days as part of Custom Label - Offboarding_PE_Delay_Days
