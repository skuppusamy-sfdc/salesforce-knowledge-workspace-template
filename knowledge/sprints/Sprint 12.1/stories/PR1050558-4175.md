## Solution

Profile “Health Cloud Restricted User - Corporate Support” (Cloned from current corporate support profile)

Permission Set Group “Health Cloud Restricted User Group”

Permission Set “Health Cloud Restricted User - Corporate Support Permission” 

- Ensure permission set has OmniStudio User permissions

*** **Read* - Flexcards (Omni UI Card), Omniscripts (Omni Process), Data Mappers (Omni Data Transformation), Integration Procedures (Omni Process), and OmniInteractionAccessConfiguration

*** **Read and Update -* OmniGlobalAutoNumber

*** **Create, Read, Update, Delete -* Omniscript Saved Session

Permission Set “Health Cloud  PNM”  Only 1 permission - **Manage Health Cloud Provider Network Management**

Permission Set “Health Cloud Restricted User  Health Cloud Platform”  Only 1 System permission -  **Manage Health Cloud**

Public Group “Manage Cases”

+Field Level Security+

Give view to all fields on all objects with read access 

Give edit access to all fields on all objects with create/read/edit access [DO NOT GIVE EDIT ACCESS TO “TERM SHEET STATUS” ON CONTRACT]
