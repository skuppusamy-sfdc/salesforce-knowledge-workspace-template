## Description

**Description:** 

Given I am Corporate Support user / Recruiter,

When I configure rates  using Contract request guided flow.

then, when i select Employee Type = Employee & Time Type = Full Time / Part Time. 

When i go to the next screen of “Facility selection”  I see HPF’s are filtered based on **Employee type only** & Not based on Time type. 

**Impact:** Contract is created based on Time Type we selected in the Guided flow, But the contract can contain Part Time HPF as well.

**Suggestion:** To add time type field on the facility selection table. so that User is aware which HPF is being selected in case of multiple HPF..

!image20251217075458.png|width=667,alt="image20251217075458.png"!

**Observation 2:** No indication on Confgure rate screen to show where its part time or Full time rate as we are not filtering rates based on HPF’s Time type.

!image20251217162537.png|width=890,alt="image20251217162537.png"!

**Suggestion:** 

IF clinician is working Part time in Facility, Then show part time rate on Rate detail screen. 

IF clinician is working Full time in Facility, Then show full time rate on Rate detail screen.
