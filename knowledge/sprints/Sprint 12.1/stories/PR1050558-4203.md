## Description

As a SCP Corporate Support user, I need to easily see a list of clinicians associated with a specific service account, so that I can view a holistic list of person accounts with active contract records

## Acceptance Criteria

Given I am a Corporate Support user

When I navigate to a Service Account

Then I am able to view:

- Clinician Contracts

** Object: Contract (associated to Service Account via Provider Network Contract)

** Filters: Status does not equal Canceled

** Columns to Display: Contract Number, Clinician Name, Employment Type, Status, Contract Start Date, Contract End Date

** Display 20 records per page

** Enable sorting for Clinician Name, Status, and Start Date column

- Healthcare Practitioner Facilities

** Object: Healthcare Practitioner Facilities (associated to Service Account via Account lookup)

** Filters: Status does not equal Inactive

** Columns to Display: Practitioner Facility Name, Practitioner, Status, Facility Title, Time Type

And the lightning record page is updated so that there’s a header, small left column, and large center column (template = header and right sidebar)

## Solution

Clinician Contracts under Facility

Create a Flex Card similar to ClinicianContract_DisplayRelatedHPF

Query: SELECT Id, Contract.ContractNumber, Contract.Employment_Type__c, Contract.Status, Contract.StartDate, Contract.EndDate FROM ProviderNetworkContract WHERE Healthcare_Practitioner_Facility__r.AccountId = '0017x0000176IKHAA2' AND Contract.Status != 'Canceled'

Make the ContractNumber a hyperline to the Contract 

Write an APEX code for sorting feature as Omni support is limited for Dynamic Sorting

Healthcare Practitioner Facilities

Use the Standard Related List for Showing Related Date using HealthcarePractitionerFacility.AccountId
