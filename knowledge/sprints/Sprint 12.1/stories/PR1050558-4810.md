## Description

When a EE clinician contract is created with a HPF (Employment Type = Employee) having Corporate Entity A, then even HPFs having Corporate Entity A but Employment Type = Independent Contractor is getting added to the EE clinician contract

Same issue is observed when adding HPF using ‘Add HPF’ button on Summary tab

**Steps to reproduce:**

1. Login to Core HC org as Recruiter - Myrtle Will

1. Open a clinician record with EE and IC type HPF records having same ‘Contracting Corporate Entity’

1. Using Contract Request button, create a clinician contract record of Employment Type ‘Employee’ choosing a HPF having Contracting Corporate Entity A

**Expected Result:**

Only HPFs having same Corporate Entity A but Employment Type = Employee should be added to the EE clinician contract

**Actual Result:**

HPFs having Corporate Entity A but Employment Type = Independent Contractor is getting added to the EE clinician contract
