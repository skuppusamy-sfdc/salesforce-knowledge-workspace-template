## Description

When using the **Add HPF** button on the clinician contract summary to add a new facility with a different NPPA tier, the system prompts the user to confirm the NPPA tier. However, the system does not allow the user to select an NPPA tier other than the value associated with the selected facility in the table.

It is now asking when there is 1 HPF in the table (Facility Selection screen). but not allowing to proceed further.

**Expectation:** System should check NPPA tier value with contracts NPPA & ask user to select NPPA tier if its mismatch between selected HPF’s  & Contract’s NPPA Tier.

## Root Cause

The original implementation did not cover this scenario. The logic only validated the NPP Tier available on the Add HPF page, instead of validating against the Contract NPP Tier. As a result, the system allowed users to select only the NPP tiers present on HPF records.
