## Description

As a Corporate Support User with access to view Risk Review cases (user has Public Group = ‘Clinician Risk Review’), I want the ability to view the related Risk Review Assessment response, so that I can see the result of the risk review.

## Acceptance Criteria

Given I am a Corporate Support User within the Corporate Support Apps

And I have Public Group = ‘Clinician Risk Review’ (access to Risk Review cases)

When I navigate to a Risk Review case (RT: Risk Review)

Then I can view the related Risk Review Assessment ('Assessment' field on case)

And Assessment Question Responses on the ‘Responses’ tab.

## Solution

Grant Access based on AC

1. Create new Picklist value at Type field on Assessment object

## In **ClinicianRiskReviewOS** OS, while creating Assessment, populate type field as **Risk Review, refer  Competency_SaveAssessmentResponse IP**

## Create a sharing rule on Assessment with below filter

### Type equals **Risk Review**

### Share with **Clinician Risk Review** public group
