**Steps to Reproduce:**

"Step 5 is where the issue occurred. 

1. On the Client Contract, the following fields are updated:

“Activated Date” with current/today’s date

“Activated By” with the user who initiated ‘Activate Contract’

“Status” = “Executed”

1. Upon Status moving to "Executed", if PMT Project field is blank on Opportunity, a Task is created for the Implementation Manager to create the PMT Project

1. Upon Status moving to "Executed", if PMT Project is populated, no Task(s) are created

1. User can add a File

1. Upon move of Contract Status to "Executed" for ALL Client contracts related to the Opportunity, then the Opportunity moves to Stage = "Implement" (Closed/Won, 100%, Closed)

1. Contract is added to the Client Contract related list"

**Expected vs. Actual Results:**

Expected results did not occur. Opportunity remained in "Prospecting" status.
