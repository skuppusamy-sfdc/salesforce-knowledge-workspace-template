## Description

**Steps to Reproduce:**

1. Log in to the Corporate Support application as an MSL user/System Administrator.

1. Navigate to Contracts tab and open any clinician record.

1. Click on list view on top right corner and observe the Activate button is available

**Expected Result:**

- Activate button should not be displayed

- User should not see or be able to perform the Activate action

**Actual Result:**

- Activate button is displayed

- User is presented with an action that should not be available.
