## Description

As a Corporate Support user with access to Practice Operations Contracts, when a Practice Operations contracts is in Status = Executed, then ‘NPPA Tier’, ‘Contract Start Date’ & ‘Contract End Date’ fields should be read only, so that we prevent users from making changes that can disrupt downstream processes and ensure that the user following the process to proper guided maintenance process (Request Contract Change) for updates.

## Acceptance Criteria

Given I am a Corporate Support user with access to Practice Operations Contracts

When I am on a Practice Operations contract in Status = Executed (including: Superseded, Abandoned, & Expired)

Then the following fields should be read only:

1. NPPA Tier

1. Contract Start Date

1. Contract End Date

## Solution

Validation Rule: in Contract for PO Contract

When Contract Status = Executed make field read as per AC

Validation rule should by pass for admins
