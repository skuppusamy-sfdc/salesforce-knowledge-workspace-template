## Description

Per story 3343, when the legal team member (Jo March) amends a Client Contract via “Amend Contract”, then the system shall clone everything except: Key Dates and Contract Name & Description

**Issue**: the Child Contracts (Contract Type ≠ Master) are not getting cloned.

## Solution

Use the correct ParentContractId field in the Amend Contract flow so the system can identify and clone all child contracts properly.

## Root Cause

Child Contracts were not cloned because the flow was filtering on the wrong parent field (`APXT_Redlining__Contract_Family_Parent__c`) instead of using the actual `ParentContractId`, causing the system to miss all child records.
