## Description

As a MuleSoft Integration Developer, I need Salesforce Change Data Capture (CDC) to trigger data transmission to MuleSoft during appropriate times, so that I can perform necessary actions

Given a Clinician Contract is executed

Then CDC Channel Contract_Trigger__chn should fire (already existing, needs update)

And Hire Date is no longer considered

Given a Clinician Contract is executed AND Hire Date is populated

Then Contract_HireDate_Trigger__chn should fire (new CDC channel)

## Solution

**Update** the condition for CDC Channel Contract_Trigger__chn

filterExpression: (Status = &apos;Executed&apos;) 

Create a **new** CDC Channel Name: Contract_HireDate_Trigger__chn

filterExpression: (Status = &apos;Executed&apos; AND Hire_Date__c != null AND (Employment_Type__c = &apos;Employee&apos; OR Employment_Type__c = &apos;Independent Contractor&apos;))

 

Make sure the channel name don't change from Solution
