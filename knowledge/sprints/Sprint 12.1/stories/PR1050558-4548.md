## Description

Steps to Reproduce:

1-Login as Legal Team member and navigate to the Client Contract record.

2-Ensure status of Client Contract is Draft.

3-Enter the Contract start date and Contract end date in Past.

4-Click on Save and an error message is displayed.

Issue - On Client Contract upon entering start and end date in Past, an error message is displayed.

Expected Result -  Error message shouldn’t be displayed.

Actual Result - On Client Contract upon entering start and end date in Past, an error message is displayed.
