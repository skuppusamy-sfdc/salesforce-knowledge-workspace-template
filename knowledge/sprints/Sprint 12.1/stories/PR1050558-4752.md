## Description

When using Add HPF button on clinician contract’s summary to add a new facility with a different NPPA tier, the system does not prompt the user to confirm the NPPA tier. The calculation is performed based on the tier selected in the contract while creating the contract.

It is asking when there are 2 HPF in the table (Facility Selection screen). but not when there is only 1 HPF with different NPPA than that  contract itself.

**Expectation:** System should check NPPA tier value with contracts NPPA & ask user to select NPPA tier if its mismatch between selected HPF’s  & Contract’s NPPA Tier.
