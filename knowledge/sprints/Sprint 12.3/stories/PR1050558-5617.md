## Description

**Description:**

System is creating duplicate HealthcareProvider records..We are not supposed to create HealthcareProvider for clinician if one already exists

select PractitionerId, count(Id) from HealthcareProvider  group by PractitionerId

HAVING COUNT(Id) > 1

limit 1000 (edited)

## Solution

Remove the logic for creation of HealthcareProvider record from Healthcare Practitioner Facility After Insert Flow (`SCP_Healthcare_Practitioner_Facility_After_Insert_Flow.flowmeta.xml`) and also Account After Insert Flow (`Account_After_Insert_Flow.flowmeta.xml`) should fail and throw the error when HealthcareProvider is not created

## Root Cause

Duplicate logic for creation of HealthcareProvider record from Healthcare Practitioner Facility After Insert Flow

Coding: Logic Errors
