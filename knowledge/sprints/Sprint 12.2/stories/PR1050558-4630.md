## Description

As a Clinician Contracting Specialist, when I select “Refresh Client Rates” on a Draft Clinician Contract, I want the system to match and update/remove rates based on the attached matching criteria, so that I am working with the most updated Client rates.

## Solution

Note: Facility Title is one of the KEY but we are processing One Facility Title at a time so that key is not included in the solution and handled at step step 1.a.ii

When the user clicks the “**Refresh Client Rates**”

Apex Code

1. For Each ProviderNetworkContract Record in the Clinician Contract

## Get the latest +relevant+ Practice Operation Contract 

### ProviderNetworkContract.Healthcare_Practitioner_Facility__r.AccountId = [Contract.AccountId|http://Contract.Id] and [**Contract.**|http://Contract.Id]**StartDate <= ProviderNetworkContract.Contract.StartDate**  AND  [Contract.|http://Contract.Id]RecordType= Practice Operation and [Contract.|http://Contract.Id]Status = “Executed” Order By [Contract.|http://Contract.Id]StartDate DESC LIMIT 1

### Get all the **Practice Operation** PreventiveCareAgreement record where ContractPaymentAgreement.Type__c = Stipend OR Shift OR Variable OR Target OR Leadership Incentives and Facility_Title__c = ProviderNetworkContract.Healthcare_Practitioner_Facility__r.Facility_Title__c. Loop through each records 

#### For ContractPaymentAgreement.Type__c = “Shift” 

##### use When Is_Deleted__c = false use [PreventiveCareAgreement.Shift__r.Shift_Full_Name__cPreventiveCareAgreement.Shift__r.Type__cPreventiveCareAgreement.Unit_of_Measure__c-PreventiveCareAgreement.Time_Type__c], [PreventiveCareAgreement.Rate__c], [[PreventiveCareAgreement.Id|http://PreventiveCareAgreement.Id]] (use a List<String, Map<Id, Number>>) **[Shift Active Records]**

##### use When Is_Deleted__c = true then use  [PreventiveCareAgreement.Shift__r.Shift_Full_Name__cPreventiveCareAgreement.Shift__r.Type__cPreventiveCareAgreement.Unit_of_Measure__c-PreventiveCareAgreement.Time_Type__c], [PreventiveCareAgreement.Rate__c], [[PreventiveCareAgreement.Id|http://PreventiveCareAgreement.Id]] (use a List<String, Map<Id, Number>>) **[Shift Deleted Records]**

#### For ContractPaymentAgreement.Type__c != “Shift”

##### use When Is_Deleted__c = false use [PreventiveCareAgreement.Type__cPreventiveCareAgreement.Unit_of_Measure__c], [PreventiveCareAgreement.Rate__c], [[PreventiveCareAgreement.Id|http://PreventiveCareAgreement.Id]]  (use a List<Id-String, Map<Id, Number>>) **[Other Type  Active Records]**

##### use When Is_Deleted__c = true use [PreventiveCareAgreement.Type__cPreventiveCareAgreement.Unit_of_Measure__c], [PreventiveCareAgreement.Rate__c], [[PreventiveCareAgreement.Id|http://PreventiveCareAgreement.Id]] (use a List<Id-String, Map<Id, Number>>) **[Other Type Deleted Records]**

## Get all the **Clinician Contract** PreventiveCareAgreement record where Cloned_From__c != null AND ContractPaymentAgreement.Type__c = Stipend OR Shift OR Variable OR Target OR Leadership Incentives  and Loop through each records

### Process Shift Records

#### For **[Clinician]** ContractPaymentAgreement.Type__c = “Shift” use PreventiveCareAgreement.Shift__r.Shift_Full_Name__cPreventiveCareAgreement.Shift__r.Type__cPreventiveCareAgreement.Unit_of_Measure__c-PreventiveCareAgreement.Time_Type__c, PreventiveCareAgreement.Rate__c compare the Rates from Step 1.a.ii.1 and when the [Practice Operation].PreventiveCareAgreement.Rate__c > **[Clinician]**PreventiveCareAgreement.Rate__c then update the **[Clinician]**PreventiveCareAgreement.Rate__c **(use  [Shift Active Records] Map)**

#### If there are any shift that is not available in the Practice Operation Contract then delete the Shift from Clinician Contract **(Use  [Shift Deleted Records])**

#### Update the **[Clinician].CPA.**Facility_Rate__c to the [**Practice Operation.CAP**].Id  **with new Id**

#### Update the **[Clinician].PCA.**Facility_Rate__c to the [**Practice Operation.PCA**].Id **with new Id**

### Process Other Records

#### For **[Clinician]** ContractPaymentAgreement.Type__c != “Shift” use PreventiveCareAgreement.Type__c-PreventiveCareAgreement.Unit_of_Measure__c, PreventiveCareAgreement.Rate__c compare the Rates from Step 1.a.ii.2 and when the **[Practice Operation]**.PreventiveCareAgreement.Rate__c > **[Clinician]**PreventiveCareAgreement.Rate__c then update the **[Clinician]**PreventiveCareAgreement.Rate__c **(Use  [Other Type Active Records])**

#### If there are any PreventiveCareAgreement.Type__c-PreventiveCareAgreement.Unit_of_Measure__c that is not available in the Practice Operation Contract then delete from Clinician Contract **(Use  [Other Type Deleted Records])**

#### Update the **[Clinician].**Facility_Rate__c to the [**Practice Operation .CPA**].Id **when its same** 

#### Update the **[Clinician].PCA.**Facility_Rate__c to the [**Practice Operation.PCA**].Id **with new Id**
