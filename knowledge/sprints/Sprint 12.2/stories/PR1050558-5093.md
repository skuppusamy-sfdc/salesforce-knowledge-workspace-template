## Description

As a Payroll team member, I need to see Time Type when viewing Pay Codes, so that I can properly distinguish between full time and part time NPPA for each shift

## Acceptance Criteria

Given I am a Payroll team member

When I navigate to a **Preventive Care Agreement record**

Then I am able to see the **Time Type** field

And on the Rate Pay Code related list (Related Tab) I can see the following fields: Name, Pay Code, and **Pay Code Label**

Given I am a Payroll team member

When I navigate to a **Contract Payment Agreement record**

Then I am able to see the following fields on the Preventive Care Agreement related list: Name, Facility Title, Rate, Unit of Measure, **Time Type**

Given I am a Payroll team member

When I navigate to a **Contract record (RT: Practice Operations)**

Then I am able to see the following fields on the Rate Pay Code related list (Rate Pay Codes tab): Shift, Facility Title, **Time Type**

## Solution

Add the fields as per AC
