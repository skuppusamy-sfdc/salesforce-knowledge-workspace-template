**Description:** When selecting Generate Term Sheet, I receive an Access Denied error - I should be able to populate a term sheet to send to the candidate. When selecting Generate Term Sheet, I receive an Access Denied error.

## Solution

Go to Setup

Go to Permission Set Group

Look for Clinician Recruitng

Add below perm set in group

DocGen Runtime USer

DocGen User Ext

Save

Do the same for Clinician Contracing and IT permission set group

## Root Cause

Below permissions Sets were missing:

- DocGen Runtime USer

- DocGen User Ext
