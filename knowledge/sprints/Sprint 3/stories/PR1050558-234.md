## Description

As a SCP Corporate Support employee, I need state picklist fields to have the same values across Salesforce, so that I can see consistency across all data and avoid errors

## Acceptance Criteria

Given I am an IT team member in the Corporate Support or Corporate Support Console apps

When I open a record in Salesforce where there is a State picklist

- Business License: State

- Corporation and Tax Information: State of Incorporation

- Account (Facility/Hospital, Corporate Entity): BillingAddress, ShippingAddress, Service_State__c, State_of_Formation__c

- Compliance Rule Assignment: State

- Contract (Client): BillingAddress, ShippingAddress

Then I see the same list of state full names, specified in the Solution below

## Solution

[Configure State and Country/Territory Picklists|https://help.salesforce.com/s/articleView?id=xcloud.admin_state_country_picklists_configure.htm&type=5]
