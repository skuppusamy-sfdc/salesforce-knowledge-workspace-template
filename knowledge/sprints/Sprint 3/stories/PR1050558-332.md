## Description

As a Recruiter, I want the system to automatically assign the “Portal User” permission set when I enable a customer user, so that the Applicant will have the proper access to login to the prospect portal.

## Acceptance Criteria

Given I am a Recruiter in the Corporate Support or Corporate Support Console apps

When I “Enable Customer User” from a clinician record

Then the system shall automatically assign the “Portal User” permission set to the newly created Applicant user.

_NOTE: Confirm Permission Set assignment as a System Administrator._

## Solution

Utilize a record triggered flow for User Object to update the Permission Set

Criteria: Check for the Contact record Type is Clinician that is related to the user.
