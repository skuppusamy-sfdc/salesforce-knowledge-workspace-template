## Description

As a Recruiter, I need to be able to access the Corporate Support app in Salesforce, so that I can view the records needed for my work in a tailored view

## Acceptance Criteria

Given that I am a Recruiter logged in to the Core HC org

When I navigate to the app launcher

Then I am able to search and select the Corporate Support Standard app and Corporate Support Console app

Given that I am a Recruiter using the Corporate Support app

When I view the navigation menu

Then I am able to see the following tabs: Home, Accounts, Contacts, Opportunities, Contracts, Cases, Reports, Dashboards

_Note: Asymbl objects (Job, Job Board Postings) will be added once Asymbl package is installed_

## Solution

Create Corporate Support Console app and Corporate Support Standard app

Corporate Support Console app = console navigation

Corporate Support Standard app = standard navigation

Configure navigation rules: 

Contacts open as subtabs of Account

Opportunities open as subtabs of Account

Contracts open as subtabs of Account

Cases open as subtabs of Contact

Assign both apps to Corporate Support permission set
