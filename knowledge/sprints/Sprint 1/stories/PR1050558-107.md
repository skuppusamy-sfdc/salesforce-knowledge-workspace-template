## Description

As a Recruiter, I need to be able to view and merge duplicate provider records, so that I am able to see an accurate view of prospective clinicians

## Acceptance Criteria

Given that I am a recruiter in the Corporate Support app

When I open OR create a clinician record where any of the listed fields have a value that matches another record (**Email OR Mobile**)

Then I see an indicator that there is a potential duplicate

And I am **still able to save** and create the new provider record

Given that I am a recruiter in the Corporate Support app

When I open OR create a clinician record where any of the listed fields have a value that matches another record (**NPI**)

Then I am **not able to save** and create the new provider record

configure duplicate rules
