## Description

As a Sales VP, I want the ability to capture “Green Light for Client Outreach”, so that additional downstream processes can be triggered if an early start has been confirmed.

## Acceptance Criteria

AC1: New Green Light field on the Opportunity

Given I am a Sales VP in the Corporate Support Apps

When I navigate to an **Opportunity** record (RT: Growth Opportunity)

Then I have the ability to update new checkbox field = “Green Light for Client Outreach”

- Placement:  Section = War Room Details, below “LOI Complete”

And History tracking is enabled on new field = “Green Light for Client Outreach” 

And ONLY the a user in the Sales VP persona can edit this field

NOTE: discussed no extra validation or guided experience needed. The Sales VP updating this field will allow onboarding to begin. (onboarding activities covered in other stories).

## Solution

- DD tab = Opportunity (Growth Oppty)

** Add New field = Green Light for Client Outreach

** Update layout

- Add field visibility to all permission sets with access to the object

- Restrict field edit access to only System Admin & Sales VP users. 

*** For Sales VP users -Add FLS to EDIT to Permission set = **Manage Client BD.* We have history where we can see who edited.  If there are issues with other users besides Sales VP updating that field, then we can relook at the solution.
