## Description

As a Medical Staff Liaison - CPA, when viewing a clinician record, I want to view related Collaborative Practice Agreements so that I can see agreements related to the clinician.

## Acceptance Criteria

Given I am a Medical Staff Liaison - CPA in the Corporate Support apps

When I navigate to a Clinician record on the Related tab

Then I can see Collaborative Practice Agreement related list (above Corporation and Tax Information related list)

And fields to display: Collaborative Practice Agreement Name, NP/PA Name, NP/PAA Professional Title, Physician Name, Credentialing Status (Physician), First Scheduled Date, Relationship Status, Supervision/Collaboration Status

## Solution

Clinical is related to HPF which is Master to Collaborative Practice agreements (CPA)

Using this relationship, Get the CPAs for an Clinician, using HPF. 

Create flex card, to display the fields mentioned above ( all of which are in CPA). (component Similar to displaying facilities at contract level)
