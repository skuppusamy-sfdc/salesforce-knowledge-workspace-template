## Description

As a Recruiter, I need to use a guided wizard for my screening calls with prospects, so that I can quickly update prospect data and have prompts for items to cover on the call

## Acceptance Criteria

Given I am a Recruiter in the Corporate Support apps

When I navigate to an ATS Applicant record

Then I can select Screening Call in the header 

And a guided experience opens where I can edit/populate the following fields: 

- Interest

** Requested Start Date (ATS Applicant - date)

** Requested Hours Per Month (ATS Applicant -number 3,0)

** Shift Preferences (ATS Applicant - text, 250)

** Best Time to Contact (ATS Applicant - text, 250)

** Ties to Location (ATS Applicant - text 250)

** Personal Interests/Hobbies (ATS Applicant - text 250)

** _Note: the fields described above need to be created on the ATS Applicant object. Create a new section labeled “Screening Call - Interest” and place it below Applicant Details_

** Professional Title (Contact.Professional_Title__c)

** Practicing Specialty (HealthcareProviderSpecialty record where Primary_Board_Specialty__c = FALSE)

** Additional Specialties (HealthcareProviderSpecialty record where Primary_Board_Specialty__c = FALSE)
