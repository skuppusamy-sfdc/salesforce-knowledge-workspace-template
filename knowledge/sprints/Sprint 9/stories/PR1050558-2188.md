## Description

As a System Administrator, I need to create list views for the different types Client Contracts, so that user can view all the contracts for the 3 different Client Contract types.

## Acceptance Criteria

AC1: Update List Views

Given I am a System Administrator and other Corporate support users with access to client contracts within the Corporate Support apps.

When I navigate to the Contracts tab

Then I can see the following list views have been **updated**:

- All Client Contracts

*** Filters: Contract Record Type = **Client Contract*

*** Fields to display: Contract Number, Account Name, Status, Contract Start Date, Contract End Date, Contract Owner Alias, **Contract Record Type*

AC2:  Add List Views

Given I am a System Administrator and other Corporate support users with access to client contracts within the Corporate Support apps.

When I navigate to the Contracts tab

Then I see the following list views have been **added**: 

- All Practice Operations Contracts

** Filters: Contract Record Type = Practice Operations

** Fields to display: Contract Number, Account Name, Status, Contract Start Date, Contract End Date, Contract Owner Alias, Contract Record Type

- All Letter of Intent Contracts

** Filters: Contract Record Type = Letter of Intent 

** Fields to display: Contract Number, Account Name, Status, Contract Start Date, Contract End Date, Contract Owner Alias, Contract Record Type
