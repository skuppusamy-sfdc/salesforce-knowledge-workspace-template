## Description

As a Data Quality team member, I need to enter rates for locums clinician in Salesforce, so that pay can be reported accurately

## Acceptance Criteria

Given I am a Data Quality team member in the Corporate Support apps

When I create a **Clinician Contract** where Employment Type = Locum

Then I am able to add the following rate types **per HPF**: 

- Normal Hourly (currency, 4 digits 2 decimals) (map to Locum “Normal Hourly” Rate)

- Holiday (currency, 4 digits 2 decimals) (map to Locum “Holiday” Hourly Rate)

- HM On Call (currency, 4 digits 2 decimals) (map to Locum HM Flat “On Call” Rate)

- HM Rounding (currency, 4 digits 2 decimals) (map to Locum “HM Rounding” Hourly Rate)

- Travel Hourly (currency, 4 digits 2 decimals) (map to Locum Additional “Travel Hourly” Rate)

- Shifts Requested (number, 3 digits 0 decimals) (map to Monthly ‘Shifts Requested”)

_Note: at a client level, locum clinicians have different rates, no need to store at client level_

_Note: Shift Requested, unit of measure will display as “per Month” & on the UI, it will not show the “$” sign._

Note: On the Rate Details step for Type of Rate = Locum we will add to the table a column to show the “# of Shifts”

## Solution

NOTE:

Contract > CPA(n) > PCA (n)

Contract  PNC (Junction)  HPF

Contract has 1 to many CPA . CPA has 1 to many relationship with PCA

CPA record has PNC from which we can know the HPF for the Contract.

Add new Code Set (Locum) and load new Health Cloud Records. 

If Contract.Employee Type : Locums is selected,

When Configure Rates for the HPF is selected, 

Display the fields mentioned in AC.

Display the Code Set **Code Set (Locum)**

On submit, create the following records with mapping

**Contract Payment Agreement.**Type__c → Locums

**Create PreventiveCareAgreement record for each of the rate type filled**

Monthly Shifts Requested → Shifts_Requested__c

Rate → Rate__c

Unit_of_Measure__c → Default to Per Hour // per hour applies to rate, and per Month for Monthly Shifts requested

Type__c - > Filter CodeSet.CodeSetType = Locum
