## Description

**Steps to reproduce:**

1. Login as SFO - Richard Caramel

1. Navigate to Corporate Support and open a Practice Operations contract (Status = Draft) of a service account. Service Account must have all team members required for approval and 2 users in VP Operations team role

1. Submit the contract for approval

1. Complete the approval as Performance Analyst, Ops Finance, Client Contracting Queue user

1. Verify if the contract is submitted to the last added VP operations user for approval

**Expected Result:**

1. Contract should be submitted to the last added VP operations user for approval

1. Approvers with Edit access to Contract object are able to edit the contract even when it is locked

**Actual Result:**

1. Contract is not submitted to the last added VP operations user for approval

1. Approvers (except Ops Finance) with Edit access to Contract object should not be able to edit the contract when it is locked
