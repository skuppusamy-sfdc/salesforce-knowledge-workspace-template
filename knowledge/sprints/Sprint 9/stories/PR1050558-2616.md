## Description

As a Corporate Support user, I need to see Clinical Match, Exit Interview & Offboarding cases on a clinician record, so that I can quickly understand important data for this person to complete appropriate actions

## Acceptance Criteria

Given I am a Corporate Support user

When I navigate to a clinician record

Then I am able to see the following additional related lists on the **Case tab:**

- Open Clinical Match 

** Filter: Record Type = Clinical Match, Status ≠ Closed

** Columns: Subject, Healthcare Practitioner Facility, Priority, Status, Owner, Last Modified Date

- Open Exit Interview 

** Filter: Record Type = Exit Interview, Status ≠ Closed

** Columns: Subject, Healthcare Practitioner Facility, Priority, Status, Owner, Last Modified Date

- Open Offboarding 

** Filter: Record Type = Clinician Offboarding, Status ≠ Closed

** Columns: Subject, Healthcare Practitioner Facility, Priority, Status, Owner, Last Modified Date

## Solution

On Clinician record page > Cases sub tab > Add a single related list of Cases ( where record type = Clinical Match and Status != Closed) . Position this related list above the exit Interview cases.

Remove existing “Offboarding Case” related list.
