## Description

**As a** MuleSoft Integration Developer, **I need** Salesforce Change Data Capture (CDC) to trigger data transmission to MuleSoft **only when a Contract's Status becomes 'Executed' AND 'Hire Date' is populated for Contract's** where Employment Type = Employee or Independent Contractor, for the rest of Employment Type **Contract's Status becomes 'Executed'** **so that** downstream systems receive contract data at the precise, relevant stages of the contract lifecycle, ensuring data accuracy and efficient processing.

## Acceptance Criteria

Update to add Clinician record type and Administrative Start Date

## Solution

Add the condition to existing CDC (for employee type IC or employee)

((emptype == ‘Employee’ OR emptype=='IC' ) && (Status == ‘executed’ and hiredate!= null) ) OR (status = executed)
