## Description

As a Recruiter, when I clone Clinician Contract rates, and I “Add HPF” for the same service account (eg. Facility Title change), then  I want the system to copy rates from the current/cloned contract to the new HPF, so that I ensure I am offering the same rate as the current contract. 

Also, I want to update the naming convention for Contract Payment Agreement records, so that we avoid duplicates when adding an HPF with the same Service Account as a cloned HPF.

## Acceptance Criteria
