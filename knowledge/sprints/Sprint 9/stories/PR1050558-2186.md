## Description

As System Administrator, I need to ensure that only specific users have access to Case record type = Clinical Match, so that information captured with the case is restricted.

## Acceptance Criteria

Given I am a System Administrator

When a user is assigned public group = ‘Clinical Match Request’

Then only these users can view Case record type = Clinical Match 

And users assigned public group = 'Manage Cases' should NOT be able to view Case record type = Clinical Match

## Solution

- Update Group Name and Label:

** From: Clinical Match Request Group

** To: Clinical Match Request

- Update existing sharing rule for 'Manage Cases'

*** Criteria: Case: Case Record Type not equal to Risk Review**, Clinical Match*
