## Description

As a Sales VP, I need to request agreements for my clients, so that we can generate a contract with necessary information for SCP to provide services

## Acceptance Criteria

AC1: Negative path - Error message

Given I am a Sales VP in the Corporate Support or Corporate Support Console apps

When I navigate to an **Opportunity** record (RT: Growth Oppty)

And Pro Forma Model Status ≠ Approved for Contracting

Then I click the “**Initiate Contracting Request**” button

And I am presented with an option to select Contract Type Request = **Client** **Agreement Request**

And an error message is displayed if I try to proceed: Pro Forma Model must be in Status = “Approved for Contracting” to submit a Client Agreement Request. 

And I can exit out of the guided experience or click cancel to close the guided experience and to return to the Opportunity record
