## Description

As a Corporate Support user I need to view Approval Work Items and Employment Type, so that I visibility approval history and can select the appropriate contract using employment type.

## Acceptance Criteria

Feedback from 9/3 onsite

- Add the ability to view Approval Work Items on the Clinician Contract record

- Update existing “Clinician Contracts” related list

** Add column: Employment Type

** !image20250903195212.png|width=1228,height=495,alt="image20250903195212.png"!

----

Given I am a Recruiter in the Corporate Support App

When I am on a **Clinician Contract** record

And I open the Approval tab

Then I have the ability to view the “Approval Work Items” (approval history) 

And can see the following:

- Approval Work Item Name

- Status

- Reviewed By

- Assigned To

- Comments

- Reviewed Date

 

Given I am a Corporate Support user with access to the Clinician Contract in the Corporate Support App

When I am on a **Clinician** record

Then I have the ability to view “Employment Type” within the existing “Clinician Contract” related list  

And I can see the following:

- Contract Number

** **Employment Type* (NEW)

- Status

- Contract Start Date (default sort with most recent at top)

- Contract End Date

## Solution

Create a flexcard - which queries the Approval Work Items (History) for the clinician and display the records (via data table or any other table format )

Add the Employment type field  to Clinician Contract related list on the Agreements subtab in clinician account record. (in the LRP)
