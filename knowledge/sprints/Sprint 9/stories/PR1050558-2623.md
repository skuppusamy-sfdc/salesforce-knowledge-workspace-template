## Description

As a HR team member, I need a notification when First Clinical Shift on a Healthcare Practitioner Facility record changes, so that I can correct dates for a clinician in Workday

## Acceptance Criteria

Given I am an HR Team member

When First Clinical Shift changes (not the first time it is populated) on a Healthcare Practitioner Facility record

Then I receive the following email notification:

- Recipient: HR@SCPhealth.com

- Subject: Change in First Clinical Shift

- Body: 

There has been a change in First Clinical Shift for [clinician name]. Please review their Healthcare Practitioner Facility record in Salesforce at the following link: [link to HPF where First Clinical Shift changed]

## Solution

Record Triggered Flow on HPF on update, when First Clinican Shift is updated/changed

Send email to the receipient mentioned above, with the details mentioned.

If there is an existing custom metadata object that holds emails, please use that.
