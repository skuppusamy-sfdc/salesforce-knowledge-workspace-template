## Description

As a Recruiter, I need the ability to see the Type details for Shift in the Rate Details step, so that I can make the appropriate selection and rate edits.

## Acceptance Criteria

Given I am a Recruiter in the Corporate Support Apps

When I navigate to the clinician contract record in Status = Draft

And on the Summary tab, I select **Configure Rates** 

And select Type of Rate = Shift

Then on the Rate Details step, in the Type field, I can see the following information displayed:

- Shift Full Name

And I no longer see the Account Name column (for all Rate Types, not just Shift)

!image20250903155306.png|width=1494,height=696,alt="image20250903155306.png"!

## Solution

For rates of type Shift, Add Client Contract’s Shift record’s full name .

For other rates types, the current type can be displayed. 

on the existing omniscript for (Configure rates), display Shift full name conditionally for shift rate type. 

Remove the Account name column from the table - for all rate types
