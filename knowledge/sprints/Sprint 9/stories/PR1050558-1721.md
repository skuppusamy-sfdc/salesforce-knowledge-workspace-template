## Description

As a Recruiter, I need to indicate that a clinician needs to start administrative work ahead of clinical work, so that the clinician is created in Workday to complete their onboarding steps in advance

## Acceptance Criteria

Given I am a Recruiter in the Corporate Support apps

When I navigate to a clinician record and I start the Contract Request process

In the Employee Type selection, if I select Employee or Independent Contractor

And I select Start New Request

In the Healthcare Facility selection, if I select HPF records where Facility Title = Assistant Medical Director, EE Asst System Medical Officer, EE Asst Hospital Med Program Director, EE Hospital Medicine Program Director, EE System Medical Officer, Interim Medical Director, Medical Director

Then I am able to populate the following checkbox field: Start Admin Work Ahead of Clinical Work

And I must indicate Administrative Start Date (populates field on Contract)

## Solution

In the Contract request omniscript, conditionally display a checkbox field “Start Admin Work Ahead of Clinical Work” on screen 3, based on the conditions

selected employee type is Employee or Independent Contractor

AND

Selected Facility (HPF.FacilityTitle__r.Code_set__r.Title__c) is Assistant Medical Director OR EE Assistant System Medical Officer OR EE Asst Hospital Med Program Director or EE Hospital Medicine Program Director OR EE Lead Midlevel Provider

Map the checkbox to field on Contract.Start_Admin_Work_Ahead_of_Clinical_Work__c
