## Description

As an Medical Staff Liaison (MSL)CC, I need to access to a MSLCC queue for tasks, so that tasks can be associated to a queue.

## Acceptance Criteria

Given that I am a MSL-CC in the Corporate Support or Corporate Support Console App

When I navigate to the Tasks tab 

Then I can view tasks (record type = Default) that have been assigned to the MSL-CC Queue; not yet assigned to an individual

Then I can see the following columns: Subject, Name,  Related To, Status, Assigned To, Due Date, Priority, Last Modified Date/Time

Then I can select the task that I want to assign to myself

Then I can click Change Owner (Assigned To)

Then I can select the new owner

create Medical Staff Liaison role (see [Security Matrix|https://schumachergroup.sharepoint.com/:x:/r/teams/UnifiedPlatform-ReGenesis/Shared%20Documents/Program%20Documentation/Program%20Planning/Discovery%20Deliverables%20%26%20Documentation/Salesforce%20team%20Deliverables/Architecture/Security%20Matrix.xlsx?d=w796b8306c5314622a033076ac7c727e6&csf=1&web=1&e=0yz7ya], tab = Role)
