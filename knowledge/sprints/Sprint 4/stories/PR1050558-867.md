## Description

On Healthcare Practitioner Facility record, the ‘Temporary Inactive Reason’ field is not enabled when Credentialling Status is ‘Temporarily Inactive’

**Steps to reproduce:**

1. Log in to Salesforce Core HC org as MSL user

1. Navigate to Corporate Support app and open a Healthcare practitioner facility record

1. Verify the values for ‘Temporary Inactive Reason’ field when Credentialling Status is ‘Temporarily Inactive’

**Expected Results:**

User should be able to select a value for ‘Temporary Inactive Reason’ field when Credentialling Status is ‘Temporarily Inactive’

**Actual Results:**

User is not able to select a value for ‘Temporary Inactive Reason’ field when Credentialling Status is ‘Temporarily Inactive’

## Root Cause

Picklist field dependency was not properly set as per requirement.

## Root Cause Type

Deployment and Configuration: Incorrect Configuration
