## Description

As a SFO (Staffing Fulfillment Optimization) user, I want the ability to generate facility rates by designating the type of rate and specified rate for each Shift Rate & Facility Title combinations, so that we have clearly defined rates for each Service Account

## Acceptance Criteria

Given I am a SFO in the Corporate Support or Corporate Support Console app

When I navigate to a **Contract** record (record type = **Client**) 

And clicked the **Specify Rates** button (part of [https://salesforce.atlassian.net/browse/PR1050558703|https://salesforce.atlassian.net/browse/PR1050558703|smart-link] )

Then I am presented with a Rate Type screen to capture:

- Type of Rate (Contract Payment Agreement, field = Type)

** Required to select one from the following list of values to proceed to the next screen:

*** Stipend

*** Shift

*** Variable

*** Target

**** **Leadership Incentives*

*** Bonus

Then when I select **Leadership Incentives**, I see the Rate Details screen and I have the following option to:

- Go back to the previous screen (button = Previous)

** And if I selected go back to the previous screen (button = Previous), then user is returned to the Type of Rate screen with the previously selected value

- Populate the following to proceed:

** Leadership Incentives

*** Type

**** select Type from (required, multi-select): Director At Risk Stipend, Director Quarterly Upside Stipend, Lead NPPA Quarterly At Risk Stipend, Director Annual Retention Stipend, Director Incentive

*** Facility Title

**** select between Facility Title (required, multi-select) = Assistant Medical Director, EE Asst Hospital Med Program Director, EE Hospital Medicine Program Director, EE Lead Midlevel Provider, EE System Medical Officer, Interim Medical Director, Leased Medical Director, Medical Director, Partner Medical Director

*** Rate

**** Required, editable Currency field (16,0); user to capture whole number

**** Unit of Measure (required); per Metric, % Salary Reduction, per Monthly Tier, per Hour, per Shift, per Month

Guided Experience UI Mapping:

- “Type” picklist values defined by: 

** Field = Code Set Type = Leadership Incentives

** Field = Active = True/Checked

** Creation of Code Set is a Subtask under [https://salesforce.atlassian.net/browse/PR1050558469|https://salesforce.atlassian.net/browse/PR1050558469|smart-link]

- “Facility Title” picklist values defined by:

** Object = Code Set 

*** Field = Code Set Type = Facility Title

*** Field = Can Have Leadership Incentive? = Yes/Checked
