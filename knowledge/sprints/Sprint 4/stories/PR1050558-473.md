## Description

As a System Administrator, I need to automatically generate Healthcare Provider and Healthcare Payer Network records in Salesforce, so that business users do not need to manually create additional records to satisfy the Health Cloud data model

## Acceptance Criteria

Given I am a Corporate Support user in the Corporate Support or Corporate Support Console apps

When I **create a Clinician Person Account** record

Then a Healthcare Provider record is created with the following attributes:

- Healthcare Provider Name = full name from Clinician Person Account

- Practitioner = lookup to Clinician Person Account

Given I am a Corporate Support user in the Corporate Support or Corporate Support Console apps

When I navigate to a Healthcare Provider record

Then I am able to see the following:

- Header - display Name only

- Main region

** Tabs component: Details

** Details tab displays fields as specified in data dictionary

Given I am a Corporate Support user in the Corporate Support or Corporate Support Console apps

When I **create an Account (record type = Corporate Entity)** record

Then a Healthcare Payer Network record is created with the following attributes:

- Payer Network Name = Corporate Entity Name

- Payer = lookup to Corporate Entity Account

Given I am a Corporate Support user in the Corporate Support or Corporate Support Console apps

When I navigate to a Healthcare Payer Network record

Then I am able to see the following:

- Header - display Name only

- Main region

** Tabs component: Details

** Details tab displays fields as specified in data dictionary

Given I am a System Administrator 

When I am in the Corporate Support or Corporate Support Console apps

Then I should have access to the Healthcare Provider and Healthcare Payer Network objects as specified in the Security Matrix (permission sets tab)

## Solution

Create a Record Triggered Flow for new Record Creation running in System Mode to auto create the records

Update **Permission Sets** for the Healthcare Provider & Healthcare Payer Network objects per Security Matrix:

- System Administrator

** Healthcare Provider: Read, Create, Edit, Delete, View All, Modify All

** Healthcare Payer Network: Read, Create, Edit, Delete, View All, Modify All

- Manage Facilities, Service Lines, and Client Contacts

** Healthcare Provider: Read

** Healthcare Payer Network: N/A

- Manage Clinicians

** Healthcare Provider: Read

** Healthcare Payer Network: N/A

- Manage Clinician Identifiers, Credentials, Specialties

** Healthcare Provider: Read

** Healthcare Payer Network: N/A

- Manage Clinician to Facility Relationships

** Healthcare Provider: N/A

** Healthcare Payer Network: Read

- Manage Clinician Contracts

** Healthcare Provider: Read

** Healthcare Payer Network: Read

Update **Permission Set Groups**:

- Staffing Operations

** Add Permission Set: Manage Client Contract

- Legal

** Add Permission Set: Manage Clinicians

- Data Quality

** Add Permission Set: Manage Clinicians, Manage Clinician to Facility Relationships
