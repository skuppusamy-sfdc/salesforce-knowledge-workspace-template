## Description

As a Medical Staff Liaison, I need a parent onboarding case to always be created during the onboarding process, so that I can easily view all child cases under a single umbrella case

## Acceptance Criteria

Given that I am a Medical Staff Liaison in the Corporate Support apps

When a parent onboarding case is created

Then I need the following +child cases+ to be created with **clinician onboarding** case record type 

_additional criteria for creating cases are captured in the clinician onboarding sheet_

- Training & SCP University

- HR Setup

Then the following information is populated: 

- Status = Not Started

- Contact Name = clinician

- Type = SCP

- Case Origin = Manual

- Priority = Medium (unless specified in early onboarding flow)
