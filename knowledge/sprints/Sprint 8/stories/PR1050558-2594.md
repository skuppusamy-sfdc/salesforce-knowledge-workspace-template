## Root Cause

DMO is able to create the records. Here mentioned users are not able to create the records reason being, they don’t have permission to create case records. As per story we have to only focus on Manage Clinician Recruiting permission set. and the mentioned users don’t have this permission set.

In the story nothing was mentioned regarding this scenario.

As now, it has been discussed that we will show the same error screen with no access message in this scenario, which has been implemented in the solution now.
