## Description

**Steps to reproduce:**

1. Login as Recruiter - Myrtle Will

1. Open a clinician contract whose related Service account has a Practice Operations contract. Practice Operations contract should have rate values configured for Bonus

1. From Summary tab of clinician contract, click ‘Configure Rates’ button

1. Select the Type of Rate as ‘Bonus’ and click ‘Next’ button

1. Verify if the existing rate is displayed along with the new Bonus fields 

1. Make changes if needed to the existing rates and click ‘Finish’ button

**Expected Result:**

User should be allowed to save just the existing Bonus rates

**Actual Result:**

User is not allowed to save just the existing Bonus rates and error id thrown enter the value for the new Bonus fields even though the user has not selected any value in Type multi-picklist field
