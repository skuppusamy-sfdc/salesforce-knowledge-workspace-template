## Description

As a Data Quality team member, I need to complete tasks for Leased and Locums clinicians, so that Salesforce has the applicable leased acknowledgement and agreement letters for relevant clinicians

## Acceptance Criteria

Given I am a Data Quality team member in the Corporate Support apps

When a Clinician Contract is updated so that Status = Executed

And the Employment Type field on Clinician Contract = Locum

Then a task record is automatically created for each HPF on the contract with the following attributes

- Subject = Attach Locum Assignment Letter to HPF

- Record Type = Default

- Assigned To = Data Quality Queue

- Status = Open

- Due Date = 1 day from automation start

- Type = Other

- Priority = Normal

- Name = clinician record

- Related To = healthcare practitioner facility record

Create Data Quality Role - reports to COO

Create Data Quality Queue - queue members are users in the Data Quality Role

## Solution

Use existing record triggered flow on Contract

When status = Executed and  Contract > Employee Type = Locum 

For each of the HPF, create task with details given above.

Use Decisio matric for task if needed
