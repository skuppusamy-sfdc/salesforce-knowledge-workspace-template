## Description

As a Clinician Contracting Specialist, I need to view the Healthcare Practitioner Facility records associated with a contract, so that I can quickly generate a contract with the facilities are relevant for a clinician

## Acceptance Criteria

Given I am a Clinician Contracting Specialist in the Corporate Support apps

When I navigate to a Contract record where Record Type = Clinician

Then I am able to see the following: 

- Header

- Main Region

** Tabs: Details, Facilities, Related, Summary, Files

*** Details _(existing, no changes)_

*** Facilities

***** **Display a list of Healthcare Practitioner Facility record related to this Contract record (via Provider Network Contract object)*

***** **Fields to display: Name [link to HPF], Account, Contracting Corporate Entity, Facility Title, Status, Credentialing Status*

****** **Clicking on the Name attribute in a row will navigate a user to the selected HPF record*

*** Related _(existing, no changes)_

*** Summary _(existing, no changes)_

*** Files _(existing, no changes)_

## Solution

Contract (Master)

- Provider Network Contract(Detail) – Has lookup to HPF

On Clinician Contract lightning page, edit and add sub tab, with custom label “Facilities”

Use the ProviderNetworkContract, to get the Healthcare_Practitioner_Facility__c records where ContractId matches current contract record

Create a flex card / Repurpose flex card ( if feasible)

Display the fields mentioned (refer DD)

AccountId, Contracting_Corporate_Entity__c, Facility_Title__c, Status__c, Credentialling_Status__c
