## Description

As a Data Quality team member, I need an automated task created when IC clinicians complete their intake form, so that I can have a reminder to review the information provided for accuracy and completeness

## Acceptance Criteria

Given I am a Data Quality team member in the Corporate Support apps

When an intake form is completed for a clinician with a HPF record where Employment Type = Independent Contractor and Status does not equal Inactive

Then I am able to see that a task has been automatically created with the following information: 

- Subject = Review Corporation and Tax Information

- Record Type = Default

- Assigned To = Data Quality Queue

- Status = Open

- Due Date = 1 business day from automation start

- Type = Other

- Priority = Normal

- Name = clinician record

- Related To = assessment record (intake form)

## Solution

Record triggered flow on Intake form, where Employment Type = Independent Contractor and Status does not equal Inactive, create a task with the details mentioned above.

Subject = Review Corporation and Tax Information

- Record Type = Default

- Assigned To = Data Quality Queue

- Status = Open

- Due Date = 1 business day from automation start

- Type = Other

- Priority = Normal

- Name = clinician record

- Related To = assessment record (intake form)

Use Task creation Decision matrix, (referred in PR1050558-559) to fill the Task details

Create Decision Matrix that has the following columns

Name: Task Creation Decision

CaseSubject (Input, Text)

TaskSubject(Output, Text)

RelatedToType (Output, Text)

RelatedTo (output, Text)

AssignedToType (Output, Text)

AssignedTo (Output, Text)
