## Solution

As part of the guided offboardidng omniscript, create cases

Check if there are existing Case (of record type = Clinician Offboarding and Type = Facility) for the HPF record.

If NO, then create Case

Subject = "Facility Offboarding-<Account Name>"

Owner = Clinician experience Specialist from Account Team Role

One child case needs to be created per the sheet - Clinical Match

-In the same flow, **refer the sheet** and create other child cases per the conditions specified-

-For Contract Review child case,- 

-When Clinician. Employment Type = Employee,  Assign to HR Queue-

-All others, Transition Specialist Queue- 

-Case.Healthcare_Practitioner_Facility__r.Facility_Title__r.Code_Set_Sub_Type__c = ‘Employee’-

Do not create Claims Closeout and Facility-Level Access Removal child case

New fields to be created : Rehirable  To be added to DD [~accountid:712020:b8b82845166549bbbced-ce96d04329bd] 

New Queues to be created :  HR Queue, Transition Specialist Queue

Use the guided flow of offboarding and create the parent case based on the sheet, 

where Contact.Status__c = Inactive

Query for Cases where ContactId = currentRecordId , Record Type = Clinician Offboarding, Type = SCP

If any record is found - do nothing

If no records found, then 

Create Case

Subject = ‘SCP Offboarding’

Related To = ContactId (clinician)

Assigned = See [sheet|https://docs.google.com/spreadsheets/d/14-PPkb2n7X13nbwyjKCH4jGUWAdaFpmlXYkoM_Rm2MM/edit?usp=sharing]
