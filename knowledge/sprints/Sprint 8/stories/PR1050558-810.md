## Description

As a System Administrator, I need to automatically generate Healthcare Provider records in Salesforce, so that business users do not need to manually create additional records to satisfy the Health Cloud data model

## Acceptance Criteria

Given I am a Corporate Support user in the Corporate Support or Corporate Support Console apps

When I **create a Service Account** record (Account, record type = Service Account)

Then a Healthcare Provider record is created with the following attributes:

- Healthcare Provider Name = name from Account record

- Account = lookup to Account record
