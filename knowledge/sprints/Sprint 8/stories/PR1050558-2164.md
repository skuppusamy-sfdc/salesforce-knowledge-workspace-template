## Description

**Steps to reproduce:**

1. Login as Recruiter - Myrtle Will

1. Open a Clinician contract record with start date (17 Aug 2025) greater than Client contract’s CPA Start Date (16 Aug 2025 11:45 pm)

1. On the clinician contract, try to configure rates for the CPA mentioned in previous step

**Expected Result:**

User should be able to see the rate from the Client Contract CPA in clinician contract

**Actual Result:**

User sees message ‘No available rates’ even though CPA Client date is less than clinician start date. If the CPA Start Date is made 16 Aug 2025 7 pm, user is able to configure the rate
