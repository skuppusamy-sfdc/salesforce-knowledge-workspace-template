## Description

As an Integration User, I need access to bootcamp and proctor attributes, so that I can create training records in Salesforce

## Acceptance Criteria

Given I am an Integration User

Then I can access the following fields: 

- Object - specific to NP/PAs, can complete all or part of the items listed below 

** Bootcamp Completion Date - new grad who hasn’t worked in EM OR a student, typically NP/PAs only, additional training through LITMOS, not normal to complete multiple times

** Bootcamp Enrollment Date - date

** Proctor Name - regional director of NP/PA services, can be specific to the site this isn’t a standard person, current state is a lookup to user object

** Proctor Site - connection to HPF, today is free text and can list multiple, future state only need one “anchor”

** Service Line - pull from Account on HPF record

** Proctorship Comments - free text to add more details, anyone able to add

** Proctorship Program Complete - Proctorship Program is the same as bootcamp

** Proctorship Program Participant - picklist Yes/No, are they a participant, implied in future by creating the record

** Skills Checklist Completion Date - Georgette or NP/PA team populates to document when provided passed off those items, date

** 30day Evaluation Completion Date  document when evaluation happened, date

** 140 hrs/clinical w/proctor compl date - when completed 140 hours of work with proctor

** 140 hrs/solo w/proctor compl date - when completed 140 hours of solo shifts

Create Training object and layout - dynamic display based on Type = Proctorship or Compliance as described in data dictionary

_all Salesforce fields today that integrate to QGenda Credentialing, these fields are not needed for integration to QGenda - only from LITMOS to Salesforce_

## Solution

Create a new obejct : Training. Create fields mentioned in DD ( for both Compliance and Proctorship sheets, ensure no duplicate fields are created)

Types: Proctorship, Compliance.

Use the type field to display the fields given in Compliance sheet and proctorship sheet in DD.

Create the new fields mentioned above in Training object in AC.

The fields should also be added to page layout

Access provided to Integration User
