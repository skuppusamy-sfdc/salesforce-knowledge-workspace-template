## Description

As a Recruiter, I need to specify details for bonus for a clinician, so that I can offer appropriate incentives for clinicians to join SCP

## Acceptance Criteria

Given I am a Recruiter in the Corporate Support apps

When I navigate to a clinician contract record where Status = Draft

And I select Configure Rates on the Summary tab

Then I am able to populate the following details for a Bonus Rate: 

- Type (already built, multiselect, updated in [https://salesforce.atlassian.net/browse/PR10505582430|https://salesforce.atlassian.net/browse/PR10505582430|smartlink] )

** **Payments* (picklist, required, available for all types)

** Values: 1 Payment, 2 Payments

** **Payroll Cycle* (number, 2 digits 0 decimals, required if 2 Payments is selected in Payments)

** Help Text: Months after milestone

** **Duration Months* (number, 2 digits 0 decimals, required)

** **Work Commitment per Month* (number, 3 digits 0 decimals, required)

** **Work Commitment Frequency* (picklist, required)

** Values: Shifts, Hours

** **Repayment Terms* (picklist, required)

** Values: Full, Pro Rata

- Rate (already built, no change)

- Unit Of Measure (already built, need to REMOVE for bonus only)

NOTE: new fields in bold are just for Type of Rate = Bonus
