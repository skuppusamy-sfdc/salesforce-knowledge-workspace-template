## Description

Steps to Reproduce:

1-Login as System Admin and navigate to the Clinician Record.

2-On Clinician Record, ensure HPF's are present as per visibility Conditions.

3-Navigate to highlight panel, click on the Login as Exp User button.

4-Portal will be displayed and on home page verify the Links under Helpful link section.

Issue - On basis of Visibility Condition below Links should be displayed.

(A)  Employed Information  It should be visible when HPF facility title has Employment Type - Employee and Status not equals to Inactive. 

As of now HPF facility title has Employment Type - Employee and but Status equals to Inactive. Then in that case Employed Information link should be hidden but it is still displayed.

(B)  Contractor Resources  It should be visible when HPF facility title has Employment Type - Independent Contractor and Status not equals to Inactive. 

As of now HPF facility title has Employment Type - Independent Contractor and but Status equals to Inactive. Then in that case Contractor Resources link should be hidden but it is still displayed.

(C)  Resident and Fellow Resources  It should be visible when clinician Professional Status - Resident or Fellow.

As of now, Clinician Professional Status - Practicing and still Resident and Fellow Resources link was displayed.
