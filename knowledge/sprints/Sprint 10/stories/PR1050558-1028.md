## Description

As a Clinician, I need to complete an intake form, so that I can provide necessary information to move forward in the recruiting process and it is retained for downstream onboarding processes

## Acceptance Criteria

Given I am a Recruiter in the Corporate Support or Corporate Support Console app

When I update the ATS Stage field on an ATS Applicant record to Awaiting Intake Form

And Intake assessments have been assigned

And after 3 days, if the Clinician has any Intake assessments in Status = In Progress

Then automatically send an email notification to the Clinician containing the following information: 

**Recipient**: Clinician’s Email

**Subject**: SCP Health | Tasks to Complete to Move Forward Together

**Body**: 

Hi, [First Name if NP/PA/RN] or [Dr. [Last Name if MD/DO]:

 

Just a quick reminder that you have tasks to complete to continue moving forward in the recruitment process. This should only take a few minutes. You can view and complete your tasks by following this link:

 

[Click here to view your tasks - Preferred Option: list out all the different open assessments as a clickable link. Second Option: display a link to portal, so they can view their assessments on the the home tab. **Final Design**: **_we will display a read only list of open Assessment name(s) and at the bottom we will display a link to the portal, so users can navigate to the portal and complete their assessment(s)_**]

 

Thank you for taking the time to complete these steps – we’re excited to keep the process moving with you!

 

[Recruiter Name - use recruiter from the original intake case]

[Recruiter Title]

SCP Health

## Solution

Scheduled Flow on Assessment Envelope Record

Filter for Status = in progress, Today = CreatedDate + 3 ( 3 days since the intake form was assigned to the Clincian), and bpats__ATS_Applicant__c.ATS_stage = ' Awaiting intake form'

and Assessment record has not been created yet - if so send email.

// applicant is related to clinician via bpats__ATS_Applicant__c lookup to contact.

then send email action 

If the assessment record is  found, then don’t send email

Query the list of assessments assigned to the Clinician and display as a  list (with link to the assessment record) in the email

Recruiter Title - Current user title
