## Description

As a MSL - CPA, I need to mass update Collaborative Practice Agreement records in Salesforce, so that I can quickly manage what records are required and not required in a streamlined manner

## Acceptance Criteria

**_this has been built with NP/PA Name and Physician Name showing the HPF record names and not the clinician names - this needs to be updated_**

Given I am a Medical Staff Liaison - CPA in the Corporate Support apps

When I navigate to the Case record where Type = Collaborative Practice Agreement

Then I can select the Review CPA Relationship tab and can perform the following: 

- View a list of Collaborative Practice Agreement records (20 record display with pagination) (see NP/PA then see all the Physicians)

** Filter on (Facility): NP/PA or Physician

** Sort on all fields listed

*** Fields to display: Collaborative Practice Agreement Name, **NP/PA Name**, NP/PAA Professional Title, **Physician Name*, Credentialing Status (Physician), First Scheduled Date, Relationship Status, Supervision/Collaboration Status

- Single or mass Edit “Relationship Status” & “Supervision/Collaboration Status” 

- User has the ability to save update

- Upon Save, the Collaborative Practice Agreement records shall be saved

## Solution

Create a Omni Script with Multi Select functionality Mass Update the CPA records.

Use Edit Block to display records 

NPPA Name: Collaborative Practice Agreement > NP/PA Healthcare Practitioner Facility > Practitioner ->Name

Physician Name: Collaborative Practice Agreement > Physician Healthcare Facility > Practitioner ->Name
