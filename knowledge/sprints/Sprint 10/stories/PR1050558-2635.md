## Description

As a Internal Corporate Support user with access to Account (RT: Service Account) records, I want the ability to see related Client Offboarding cases, so that I can see all the Service Accounts that are being terminated.

## Acceptance Criteria

Given I am a Internal Corporate Support user within the Corporate Support apps

And I have access to view Account (RT: Service Account) records & Client Offboarding Cases

Then on the a Account (RT: Service Account) record, on the Related tab I can see: (NO Actions available)

- Related list = Open Cases 

** Filter: Parent Case = BLANK, Status ≠ Closed

** Fields Displayed: Subject, Account Name, Case Record Type, Type, Priority, Status, Owner, Last Modified Date/Time

- Related List = Closed Cases

** Filter: Parent Case = BLANK, Status = Closed

** Fields Displayed: Subject, Account Name, Case Record Type, Type, Closed Reason, Owner, Last Modified Date/Time

And remove current Cases related list.

NOTE: 

- The new Open and Closed Cases related list are not specific to a Record Type (visibility to see the different record types are based on the user’s current access) 

- Creation of new Client Offboarding case record type is part of [https://salesforce.atlassian.net/browse/PR10505581683|https://salesforce.atlassian.net/browse/PR10505581683|smart-link]

## Solution

Create related list for Account Object with criteria specific in AC
