## Description

As an Integration User, I need to know when Healthcare Practitioner Facility are updated to Active status in Salesforce, so that I can update downstream systems

## Acceptance Criteria

Given I am an Integration User

When I update Contract Status is Updated as Executed on a Contract record Where Employment Type = Employee, Independent Contractor, Partner, Managed

Then the following updates happen automatically: 

- When Healthcare Practicer Facility (HPF) First Clinical Shift is available

- Trigger the following platform event: 

** Platform Event Fields, Name = HPFUpdates

*** HPF Id

*** Contract Id

*** Global Provider Id

*** Global Facility Id

*** Type = “Rate for HPF”

## Solution

When the Contract.Status = “Executed”

Query ALL HPF Related to that Contract Record with Condition 

HealthcarePractitionerFacility.First_Clinical_Shift__c IS NOT NULL

Send a bulk Platform Event

For Each HealthcarePractitionerFacility record create a Platform Event

- Platform Event Fields, Name = HPFUpdates

** HPF Id = HealthcarePractitionerFacility.Id

** Contract Id = Current Contract.Id

** Global Provider Id = HealthcarePractitionerFacility.Practitioner.Global_Provider_ID__c

** Global Facility Id = HealthcarePractitionerFacility.Account.Global_Facility_ID__c

** Type = “Rate for HPF”
