## Description

As a SFO, I want the ability to make Facility (Service Account) Contract changes to Rates and Coverages, so that changes can flow to downstream process such as Clinician contracting,Conga, Docusign, Qgenda, & Workday.

## Acceptance Criteria

-Given I am a SFO-

-When I am on a Service Account record-

-Then I enter the Change Request guided experience-

-Then I must select which changes are applicable:- 

- Shift or Variable Rate Increase

- Shift/Variable Rate Reduction or Structure Change

- Stipend or Leadership Incentive Rate Change

- Coverage Change (note for new user story  if coverage change, create a task for Payroll queue to review before client contract is requested)-

- Coverage Change with New Shift Type

- NP/PA Tier Change

-And I am able to select one or more changes to proceed with-

-Then I must specify a Start Date (date, required) and Description (text, 1000, required) for these changes- 

-Then the system clones the Service Account’s Practice Operations Contract in Status = “Executed” with the newly created contract in Status = Draft-
