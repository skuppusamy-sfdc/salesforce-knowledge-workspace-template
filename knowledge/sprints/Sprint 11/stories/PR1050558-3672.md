1. From Console, choose Jobs

2. Drill into the job you want to work

3. Navigate to the ATS Kanban tab

4. Choose the Applicant you want to work from the ATS Stage = Interviewing

5. Select the down arrow and choose ""Schedule Interview""

Needs to go straight to Schedule New Interview screen instead of having an error pop up.

## Root Cause

The system contained over 50,000 Contact records with specific record types (**Master, Internal Contact, SCP Internal**) that were **not correctly mapped** to the required **Contact Layout**. This configuration gap prevented the system from properly rendering the necessary visual components on the "Schedule New Interview" screen, triggering the error.

Deployment and Configuration: Incorrect Configuration
