## Description

As an Implementation Specialist or the Opportunity Owner, when I determine that client onboarding needs to begin early, I want the ability to trigger a task to be created for our team, so we know to manually create the PMT Project & Clone the Phases/Tasks which will be used to track the onboarding process for each Service Account.

## Acceptance Criteria

AC1: Manually trigger task to Create PMT Project

Given I am a Implementation Specialist or Opportunity Owner in the Corporate Support Apps

When I am on the Opportunity (RT: Opportunity Growth) record

And the Opportunity Stage = “Validate Solution” or “Negotiate Contract & Close”

And “PMT Project” field is not populated

Then I have the ability to click “Begin Client Onboarding”

And the system shall automatically create a **Task** to notify the users assigned to the “Implementation Queue” to create a new PMT Project:

- Record Type = Default

- Subject = Create a PMT Project for <Opportunity Name>

- Assigned To = Implementation Queue

- Comments = 1. Please create a new PMT Project for the Opportunity 2. “Clone Phases/Tasks” for each Service Account 3. When ready to send out the related tasks, update the PMT Task Status = “In Progress”

- Comments = 1. Create a new PMT Project for the Opportunity from PMT Template 2. Associate the Opportunity to the PMT Project record  3. “Clone Phases/Tasks” for each Service Account 4. For each PMT Phase associate the Service Account  5. When ready to send out the related tasks, update the PMT Task Status = “In Progress”

- Status = Open (default)

- Due Date =<+5 days from date of creation>

- Type = Client Onboarding

- Priority = Normal (default)

- Related To = <Opportunity (RT: Opportunity Growth)>

NOTE: if the criteria is met we will create a task, user can manually close (canceled) if they have not populated the “PMT Project” and a duplicate task is created.

## Solution

Create a new Action in Opportunity Name  “Begin Client Onboarding” is visibly only when  Opportunity Stage = “Validate Solution” or “Negotiate Contract & Close” and PMT Project = NULL

The button will create a task record as specified in the AC
