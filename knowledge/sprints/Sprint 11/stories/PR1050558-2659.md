Create a Record Triggered Flow for Shift Object Record New or Update Records:

When the Records is Created or Updated

- When the Cloned_From__c Is NULL

** Query for All Shift Records for the Contract where Type__c = CurrentRecord.Type__c AND Shift_Name__c = CurrentRecord.Shift_Name__c AND Cloned_From__c != NULL

*** If Records are returned then Existing_Shift_Type_Added__c = true, New_Shift_Type_Added__c = false

*** Else New_Shift_Type_Added__c = true,  Existing_Shift_Type_Added__c = false

- When the Cloned_From__c != NULL && Is_Deleted__c is updated

*** Query for All **PreventiveCareAgreement* Records for the Contract where Shift__c = [CurrentRecord.Id|http://CurrentRecord.Id] 

*** When Shift__c, CurrentRecord.Is_Deleted__c = true

***** Update all **PreventiveCareAgreement** records returned **PreventiveCareAgreement*.Is_Deleted__c = true

*** When Shift__c, CurrentRecord.Is_Deleted__c = false

***** Update all **PreventiveCareAgreement** records returned **PreventiveCareAgreement*.Is_Deleted__c = false
