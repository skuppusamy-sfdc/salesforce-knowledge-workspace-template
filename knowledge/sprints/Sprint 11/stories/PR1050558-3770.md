Description:  Applied to Job 00001 as an Applicant/Clinician. Got a message that the application/submission was successful.

When I'm in SF I don't see my application on job record.

1. Navigate to external job board

2. Apply to a job

3. Navigate to the associated job record in SF

4. Review Job Board Applicants tab

Would expect applicant to be visible in Job Board Applicant tab

## Root Cause

The core system's design failed to correctly handle the data exchange from the external job board and/or the subsequent logic required to ensure the newly created applicant record was immediately processed and rendered in the correct Salesforce UI component. This caused applicants to be successfully created but **invisible** in the expected tab.
