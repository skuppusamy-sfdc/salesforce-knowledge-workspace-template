- New field creation:

*** ‘Services Sold’ (multi-select) [~accountid:61f8618e8d9e3c00688e380b] can we make this a global picklist, I need it for Letter of Intent and maybe PMT Phase

*** “Client Termination Date” (date)

*** “Client Termination Time” (time)

*** “Client Termination Time Zone” 

***  “Terminated By” (picklist)

** Update FLS on the following permission sets:

****  **Manage Client Onboarding*

**** **View Client Onboarding*

Use a Screen Flow:

- Account / Hospital Name : ProviderNetworkContract.HealthcareProvider.Account.Name

- Service Line: ProviderNetworkContract.HealthcareProvider.Account.Service_Line__r.Name

- Services Sold: ProviderNetworkContract.Contract.Services Sold

- Physical Address: ProviderNetworkContract.HealthcareProvider.Account.Parent.BillingAddress

- City: ProviderNetworkContract.HealthcareProvider.Account.Parent.BillingCity

- State: ProviderNetworkContract.HealthcareProvider.Account.Parent.BillingStateCode

- Zip: ProviderNetworkContract.HealthcareProvider.Account.Parent.Zip

- Facility Short Name: ProviderNetworkContract.HealthcareProvider.Account.Parent.Facility_Short_Name__c

- Staffing Team: ProviderNetworkContract.HealthcareProvider.Account.Staffing_Team__c

- Client Division: ProviderNetworkContract.HealthcareProvider.Account.Client_Division__c

- Client Region: ProviderNetworkContract.HealthcareProvider.Account.Client_Region__c

- Client Termination Date: ProviderNetworkContract.Contract_Termination__c

- Client Termination Time: ProviderNetworkContract.Facility_Termination_Time__c

- Client Termination Time Zone: ProviderNetworkContract.Client_Termination_Time_Zone__c

- Terminated By: ProviderNetworkContract.Terminated_By__c

- Billing Company: ProviderNetworkContract.HealthcareProvider.Account.Billing_Company__c

- Corporate Entity: Query SELECT  RelatedAccount.Name FROM AccountAccountRelation WHERE AccountAccountRelation.AccountId = ProviderNetworkContract.HealthcareProviderId.AccountId  AND PartyRoleRelationId = (SELECT Id FROM PartyRoleRelation WHERE RoleName = 'Services Contracting Entity') AND StartDate =< Today() and (EndDate >= Today() OR EndDate is NULL) Order BY AccountAccountRelation.StartDate ASEC LIMIT 1

- Scribe Hours per Day: Select Scribe_Hours_per_Day__c from Contract where RecordType = Practice Operation and Status = “Executed” and StartDate < Today() and AccountId = ProviderNetworkContract.HealthcareProvider.AccountId Order by Start Date DESC LIMIT 1
