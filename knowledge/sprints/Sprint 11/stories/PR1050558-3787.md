## Root Cause

The defect in EMR information retrieval was caused by a **logic error** in the application's data lookup and matching process.

** **Logic Failure:** The original system logic, when provided with a clinician's Global ID, was **unable to accurately identify or map to the corresponding Hospital Account* when using common search parameters (such as nickname or full name). This failure prevented users from accessing the correct EMR data.

** **Resolution:** The fix involved **refining the data retrieval logic** to successfully extract the **Facility Global ID from the HPF records*. This improved process now allows the system to accurately locate the necessary hospital account and present a list for user selection, thereby enabling correct EMR data access.
