## Description

As a VP Operations (VPO), I need the ability to view Clinicians associated to the facility being terminated, so that I can determine the impact.

## Acceptance Criteria

Given I am a VP Operations (VPO) in the Corporate Support apps

When I navigate to the Facility Offboarding child case = **Clinician Review**

Then I am able see the Clinicians related to the facility being terminated (new tab = Clinician Roster)

And I am able to see the following information about Clinicians related to the Service Account (Clinician has an Active HPF with the Service Account listed on the case): 

- Name (clinician’s Name from the HPF’s Partitioner field)

- Sign On Bonus (checkbox; checked if the Clinician has an Clinician Contract (any status) in Status = “Executed” for the Service Account with a Preventive Care Agreement “Type” = “Sign On Bonus”)

- Location (state)  Mailing address  State

- Business License - State 

** Need the ability to see different states if more than one record is found. 

- Active in other Facilities (check box) - other active HPFs

NOTE: When the user (VP Operations (VPO)) is done, they will manually close their task = "Clinician Impact Meeting", by updating the task status = Completed

## Solution

Create new case record type = Client Offboarding & Layout - see DD

Read Only View: Use Flex Card or Edit Block (there is ready only)

Select All HealthcarePractitionerFacility with HealthcarePractitionerFacility.Status__c <> “Inactive”  realted to the Service Account (HealthcarePractitionerFacility.AccountId)

1. Name → HealthcarePractitionerFacility.Practitioner.Name

1. Sign On Bonus → if the query returns any result then check the box for that HPF record

##   Query All PreventiveCareAgreement**.**ContractPaymentAgreemen.Type__c = “Bonus” AND PreventiveCareAgreement**.**Type__r.Name = “Sign On Bonus” AND PreventiveCareAgreement.ContractPaymentAgreemen.Contract.StartDate + MonthsADD( Duration_Months__c ) > Case.Termination_Request_Date__c

## No need to check the PNC. Use Contract.AccountId = HPF.PractitionerId and get the PreventiveCareAgreement

1. Location (state)  Mailing address  State

## HealthcarePractitionerFacility.Practitioner.MailingState

1. Business License - State

## Query all records in Billing License for Clinician based on HealthcarePractitionerFacility.PractitionerId and show in comma separated list

1. Active in other Facilities (check box) - other active HPFs →  if the query returns any result then check the box for that HPF record

## Query HealthcarePractitionerFacility.PractitionerId = HealthcarePractitionerFacility.PractitionerId AND Status__c != “Inactive” AND  HealthcarePractitionerFacility.AccountId != (Case.AccountId)
