Description: Send a document via DocuSign from an Approved Contract Record. Status/Dates did not populate/update. Signed as the first signer (i.e. Clinician), status/dates did not populate/update.

2. Create Conga Agreement to create agreement records

3. Navigate back to the Contract Record and select DocuSign for Contracting

When a document is sent/signed/countersigned using DocuSign, dates and status on the Contract and Agreement Record(s) should be updated. Expected Results are below. Doesn't appear any updates are working.

-Contract Sent Date field is populated with appropriate date

-Agreement Status and Status Category are In Signatures, Other Party Signatures

## Root Cause

**Integration issue** between the Salesforce Contract Record and the **DocuSign** platform.

While the process successfully sends the contract via DocuSign and captures the 'Sent Date' and initial status updates, the subsequent automated process responsible for handling the DocuSign completion callback is **failing to execute its update logic correctly**.

Third-Party Integration: Dependency Issues
