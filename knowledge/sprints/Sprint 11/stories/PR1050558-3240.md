## Description

As an Implementation Specialist, I want the system to update the PMT Task “Task Completion %” field based on the related action plan template, so that I know the percentage of task that need to be completed.

## Acceptance Criteria

AC1: Auto update the PMT Task (Milestone) “Task Completion %” field

Given I am a Implementation Specialist in the Corporate Support Apps

When I updated the PMT Task “Task Status” = ‘In Progress' (AC1)

And the system auto executed the Action Plan Template

Then the system shall update the PMT Task’s “Task Completion %” field based on each Task status in the Action Plan is updated:

- ”# of Completed Tasks” divided by  “Total Tasks” to calculate the percentage (Ignore the Cancelled Task if any). 

- “No. of Completed Tasks” update the task when each Action Plan Task is completed 

And when the all the Action Plan Tasks are completed

Then the system shall update the PMT Task “Task Completion %” field = 100%

NOTE: current functionality is that if PMT Task “Task Completion %” field = 100 % the “Task Status” field is auto updated to “Completed”

## Solution

Update the record triggered flow [Task After Flow|https://corehc-dev1.sandbox.my.salesforcesetup.com/builder_platform_interaction/flowBuilder.app?flowId=3017x00001NS9PHAA1]

1. When the Task.Status is updated to Completed or Canceled or Open check if the Task is part of Action Plan by running a query on ActionPlanItem.ActionPlanId = <TaskId> AND ItemEntityType = Task

1. If this is part of an Action Plan (if Step 1 return a Query)

## ActionPlanItem.ActionPlanId is the Action Plan Id

## Look if the Status of the Action Plan is “In Progress”

## if not update the Action Plan status = “In Progress”

1. Get the number of Task Closed and Open for the Action Plan 

## Get All ActionPlanItem.ItemId records for that ActionPlanItem.ActionPlanId

## Query Task based on All ActionPlanItem.ItemId

## Count No of Task that are “Open”, “Completed” (Note: We are ignoring any cancelled task if any)

## Sum “Open” + “Completed” to calculate  Total Tasks

1. Look if the Action Plan Id is stored as part of PMT Task (inov8__PMT_Task__c.Related_Action_Plan__c) if found a match then update with below details

## inov8__PMT_Task__c.Total_Tasks__c = step 3.d

## inov8__PMT_Task__c.No_of_Completed_Tasks__c = step 3.c ( count “Completed”)

## inov8__PMT_Task__c.inov8__Percentage_Completion__c = count “Completed” / step 3.d “Total”

Check OOTB Flow Template as reference **Update Action Plan Status**
