## Description

As a Medical Staff Liaison, I need task records to be closed automatically, so that I do not have to do duplicate work in multiple systems

## Acceptance Criteria

Given I am a MSL in the Corporate Support apps

When specified fields are updated in a Healthcare Practitioner Facility record

Then the specified task record is updated so that Status = Completed

- Facility Onboarding parent case

** MSS Onboarding child case

**** Defect Analysis - Start Date: close when **Defect Analysis Start Date* is populated on the related HPF record

**** Defect Analysis - Complete Date: close when **Defect Analysis Completed Date* is populated on the related HPF record

**** Send Request & Release form: close when **Request and Release Returned* is populated on the related HPF record

**** Perform PSL2 - Start Date: close when **Prescreen Level 2 Requested* is populated on the related HPF record

**** Reference Verification Review: close when **Received Current Completed References* is populated on the related HPF record

**** Perform PSL2 - Complete Date: close when **Prescreen Level 2 Completed* is populated on the related HPF record

** Hospital Application Preparation child case

**** Send Hospital Application to Clinician: close when **Hospital Application Sent On* is populated on the related HPF record

*** COI (Certificates of Insurance) Completion: _manual close - no automation_

**** Determine First Scheduled Date: close when **First Scheduled Date* is populated on the related HPF record

**** Submit Hospital Application: close when **Hospital Application Sent Date* is populated on the related HPF record

** Hospital Privileging child case

**** Hospital Privileging: close when **Hospital Privileged Date* is populated on the related HPF record

** Payer Enrollment child case

**** Review Facility Requirements & Create PEP: close when **PEP Sent Date* is populated on the related HPF record

**** PEP Returned: close when **PEP Returned Date* is populated on the related HPF record

** Hospital Orientation child case

**** Schedule Hospital Orientation: close when **First Scheduled Date* is populated on the related HPF record

_criteria for closing tasks are also captured in the clinician onboarding sheet_

## Solution

Create/reuse record trigger flow on HPP. Look for the Case using the Case.HpfhealthcarePractitionerId  and then look for the task using CaseId and subject. 

Update Task status to Completed according to the field updates provided. 

- Testing: ok to update information manually for testing purposes
