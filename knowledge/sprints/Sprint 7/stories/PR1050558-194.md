## Description

As a Recruiter, I need to request approval for clinician compensation, so that I can offer appropriate pay and incentives for clinicians to join SCP

## Acceptance Criteria

Given that I am a Recruiter in the Corporate Support apps

When I navigate to a **Clinician** record

And clicked on the **Contract Request** button

And I completed the process for a net new contract (current contract rates NOT copied over) or Start From Previous Active Contract (as defined in [https://salesforce.atlassian.net/browse/PR10505581070|https://salesforce.atlassian.net/browse/PR10505581070|smart-link] )

And I am on **Clinician Contract’s Summary tab** ([https://salesforce.atlassian.net/browse/PR1050558894|https://salesforce.atlassian.net/browse/PR1050558894|smart-link])

And I can see all the Healthcare Practitioner Facilities (HPF) associated with the Contract filtered by Contract Employment Type and the HPF Facility Title lookup code set Employment Type. (Sakthi: Added this condition to filter the HPF) 

Then for each HPF I have the ability to click on the **Configure Rates** button

And when I select the Configure Rates button:

- Rate Selection 

** Single-select list for types of rates to configure:

**** Stipend, Shift, Variable, Target, **Bonus*, Leadership Incentives

** Navigation buttons

*** Cancel

**** Returns to Clinician Contract’s Summary tab and no records were created or updated

*** Next

- Rate Details

*** When **Bonus* was selected, I can see the rates for the selected Type of Rate:

*** If rates exist, then the user will see the existing rates displayed

*** If rates do NOT exist, the user will see pre-determined Facility Rates from the Account for the selected Healthcare Practitioner Facility record

**** System automatically filters rates based on the Facility Title in the HPF record

**** Based on Contract Start Date, filter Facility Contract Payment Agreement rates that are applicable on that Start Date.

*** User is able to de-select rates that do not apply to this clinician 

** User has the ability to add Additional Bonuses to One Facility. [Split the enforce of 1 facility to story [+PR10505582086+|https://salesforce.atlassian.net/browse/PR10505582086]]

*** Help Text: Enter the bonus to the facility at which they are primarily working.

*** Able to add additional bonuses from the following based on the Employment Type noted in parenthesis: (Employment Type comes from the HPF)

**** Sign On Bonus (Only for IC)

**** Commencement Bonus (Can be EE or IC)

**** Relocation Bonus (Can be EE or IC)

**** New Grad Bonus (Only for EE NP/PA)

**** Retention Bonus (Can be EE or IC)

**** Locums Buyout (Can be EE or IC)

**** Other Bonus (Can be EE or IC)

** Account Name, Type, Rate, and Unit of Measure

** I am able to update the Rate (only when Override Prohibited is unchecked) or leave it with the existing value ( [https://salesforce.atlassian.net/browse/PR1050558839|https://salesforce.atlassian.net/browse/PR1050558839|smart-link] )

*** User is not able to change the Rate to 0 or a negative number

** Navigation buttons

*** Cancel

**** Returns to Clinician Contract’s Summary tab and no records were created or updated

*** Previous

**** Returns to Rate Selection screen with the updates that the user made previously

*** Finish

**** Returns to Clinician Contract’s Summary tab

When I finish the guided experience

Then the system shall create the appropriate Contract Payment Agreement & Preventive Care Agreement records as specified in the solution.

## Solution

Add Health Cloud Records -[ tab Code Set (Bonus)|https://docs.google.com/spreadsheets/d/1B6WddJnXHDZD5kx5F1mZDqEvcAwYn1wCPejZ0ZgJpP0/edit?gid=287559019#gid=287559019]

Summary screen will be built in story 894 which will expose **Configure Rates** button in context of each Facility (which are part of Draft Contract). 

When user clicks Configure Rates button - launch existing OmniScript Clinician_ConfigureRates_English. 

The cloning process should work like below

1. Using the Parent Contract (ParentContractId) in Contract

1. Use the AccountId and PractitionerId and Facility_Title__c from current selection to get the same AccountId and PractitionerId and Facility_Title__c from Previous Contract (identified by ParentContractId)

## When a HPF was found in the Previous Contract Copy All **Contract Payment Agreement** with Rate Type = Bonus and all related child **Preventive Care Agreement** **_all fields copied exactly as is_**

### **Contract Payment Agreement**.Cloned From = (Parent Clinician Contract) **Contract Payment Agreement**.Id

### **Preventive Care Agreement**.Cloned From = (Parent Clinician Contract) **Preventive Care Agreement**.Id

### **Preventive Care Agreement**.Rate = (Parent Contract) **Preventive Care Agreement**.Rate 

## When a HPF was not found in earlier contract or a new Contract then copy the rates from the Client Contract **_all fields copied exactly as is_**

### **Contract Payment Agreement**.Facility Rate = (_Client Contract_) **Contract Payment Agreement**.Id

### **Preventive Care Agreement**.Facility Rate = (_Client Contract_) **Preventive Care Agreement**.Id

### **Preventive Care Agreement**.Rate = (Client Contract) **Preventive Care Agreement**.Rate
