## Description

**Steps to reproduce:**

1. Login to Core HC org as Recruiter - Myrtle Will

1. Open a client contract record with values configured for different Type of Rate

1. Navigate to Summary tab and click Delete icon near the Type of Rate header

1. Click Delete icon near the individual rates

**Expected Result:**

1. All rates should be deleted when user clicks the Delete icon near the Type of Rate header and corresponding CPA records should be deleted

1. Respective rate should be deleted when user clicks the Delete icon near the individual rates and CPA record should also be deleted if there are no other rate under the Type of Rate

**Actual Result:**

1. Rates are not deleted when user clicks the Delete icon near the Type of Rate header

1. Rates are deleted when user clicks the Delete icon near the individual rates but CPA records remain even though there is no rate associated with the Type of Rate. No PCA record is available under the CPA record

## Root Cause

Point 1 has been fixed, some logic was changed in IP may be due to development from multiple devs and for point 2 we show a message to the user for no records available
