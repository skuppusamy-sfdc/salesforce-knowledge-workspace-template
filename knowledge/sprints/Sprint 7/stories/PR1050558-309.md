## Description

As a Clinician Experience Specialist or Scheduler, I need to request shiftspecific bonus approval in Salesforce, so that I can offer appropriate incentives for clinicians to work a hardto-staff shift

## Acceptance Criteria

Given I am a Clinician Experience Specialist or Scheduler in the Corporate Support apps

When I navigate to a Healthcare Practitioner Facility record

Then I am able to select the **Open Shift APAY** button in the header

And the following displays: 

- Clinician Contracts where status = Executed and Employment Type = Employment Type related to HPF Facility Title 

** When no records are available, display the following message “No Active Contract is Available for this Clinician” 

- Shift Selection

** Shift Start Date/Time (date/time, required)

** Shift End Date/Time (date/time, required)

** Shift Duration (formula based on fields above)

** Once Start Date/Time and End Date/Time are populated:

*** Display a list of Shifts from Clinician Contracts where status = Executed and Employment Type = Employment Type related to HPF Facility Title

** _Navigation Options_

*** Next

- Incentive Details 

** Display the Shift selected in previous screen

** Facility Rate (formula based on client contract, for the selected shift)

** Clinician Rate (formula based on clinician contract, for the selected shift)

** Remaining Open Shifts (number, required)

*** Help Text: How many remain after getting this shift request approved

** % of Clinicians Contracted to Work (percentage 3 digits and 0 decimal points, required)

*** Help Text: FSR %

** Reason for Shift Bonus (picklist, required)

*** Picklist values: Abrupt Removal of Provider, Admin Reasons, Call Off, Core Provider Needed, Higher Hourly Rate, Holiday Pay Extra, New StartUp, Shift TravelExpenses, Surge/Additional Coverage, Tiered Bonus

** Bonus Amount (currency 5 digits and 2 decimal points, required)

** Bonus Unit of Measure (picklist, required)

*** Picklist values: per Hour, per Shift

** Comments (free text, required)

** _Navigation Options_

*** Cancel (“X” icon)

**** Returns to Healthcare Practitioner Facility record and no Case records were created or updated

*** Previous

**** Returns to Shift Selection screen with the updates that the user made previously

*** Save

**** Creates new Case record as specified in the solution

***** Case Record Type = Rate Approval

***** Type = Open Shift APAY

**** Returns the user to the Healthcare Practitioner Facility record
