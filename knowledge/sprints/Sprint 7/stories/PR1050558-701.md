## Description

As a MSL, I need to terminate a clinician’s employment at all SCP facilities in Salesforce, so that I can accurately update the status of employees at facilities that SCP staffs for

## Acceptance Criteria

Given I am a Medical Staff Liaison in the Corporate Support apps

When I view a **Clinician** record

Then I am able to select Initiate Termination Request and see the following: 

- Facility Selection

** I can select between HPF records where Status does not equal Inactive 

*** If all HPFs are selected, I must confirm: Full SCP Termination (checkbox)

*** If no HPFs are selected, display message: Please select at least one Healthcare Practitioner Facility

*** Fields to display: Account Name, Status, Last Date Worked

And if NO Healthcare Practitioner Facility records matching the above criteria, then display the following message: "This clinician does not have any prospect or active HPFs to be terminated"

## Solution

Use Omniscript ( where this is first screen) ,to display list of HPF records that are not inactive 

Display the records as a edit block (check box) list. 

Display the fields on HPF Name , HPF Status__c, Last_Date_Worked__c 

(Query HPF records using PractitionerId as the link to the Clinician)

If all records are selected, then display additional checkbox “Full SCP Termination”

The records that are selected are shown in the next screen (PR1050558-306)

Message if user does not select an HPF - Please select at least one Healthcare Practitioner Facility
