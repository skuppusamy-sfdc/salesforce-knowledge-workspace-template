## Description

As a user, I need the ability to view a component that highlights the status(es) of a clinician, so that I am able to understand where they are during their journey with SCP Health.

## Acceptance Criteria

Given a user of the system,

when I navigate to the Person Account Clinician record At a Glance tab,

Then I see a component with the following details:

- Titled “At a Glance” (see wireframe)

- A key/legend directly below it (see wireframe)

** Grey Circle Icon “Not Started”

** Blue + White Circle Icon “In Progress”

** Green Circle with Check Icon “Completed”

- Sections displaying “Section Header” values from the reference doc (“Status Label + Logic” tab - link below)

** Multiple Section Headers can be visible at the same time

** Sections have accordion feature (collapse / expand)

** Status/Label values (internal/external) visible nested under Section Headers only when section header is available

** Icons from the legend are available below the Status/Label value (see wireframe in the solution)

** In Onboarding and Offboarding sections, Internal is visible above Facility

Note:
