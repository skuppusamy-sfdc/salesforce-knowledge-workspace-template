## Description

As a Recruiter, I need to request approval for clinician compensation, so that I can offer appropriate pay and incentives for clinicians to join SCP

## Acceptance Criteria

Given that I am a Recruiter in the Corporate Support apps

When I navigate to a **Clinician** record

And clicked on the **Contract Request** button

And I completed the process for a net new contract (current contract rates NOT copied over) or Start From Previous Active Contract (as defined in [https://salesforce.atlassian.net/browse/PR10505581070|https://salesforce.atlassian.net/browse/PR10505581070|smart-link] )

And I am on **Clinician Contract’s Summary tab** ([https://salesforce.atlassian.net/browse/PR1050558894|https://salesforce.atlassian.net/browse/PR1050558894|smart-link])

And I can see all the Healthcare Practitioner Facilities (HPF) associated with the Contract

Then for each HPF I have the ability to click on the **Configure Rates** button

And when I select the Configure Rates button:

- Rate Selection

** Single-select list for types of rates to configure:

**** Stipend, Shift, **Variable*, Target, Bonus, Leadership Incentives

** Navigation buttons

*** Cancel

**** Returns to Clinician Contract’s Summary tab and no records were created or updated

*** Next

- Rate Details

*** When **Variable* was selected, I can see the rates for the selected Type of Rate:

*** If rates exist, then the user will see the existing rates displayed

*** If rates do NOT exist, the user will see pre-determined Facility Rates from the Account for the selected Healthcare Practitioner Facility record

**** System automatically filters rates based on the Facility Title in the HPF record

**** Based on Contract Start Date, filter Facility Contract Payment Agreement rates that are applicable on that Start Date.

*** User is able to de-select rates that do not apply to this clinician

** Account Name, Type, Rate, and Unit of Measure

** I am able to update the Rate (only when Override Prohibited is unchecked) or leave it with the existing value

*** User is not able to change the Rate to 0 or a negative number

** Navigation buttons

*** Cancel

**** Returns to Clinician Contract’s Summary tab and no records were created or updated

*** Previous

**** Returns to Rate Selection screen with the updates that the user made previously

*** Finish 

**** Returns to Clinician Contract’s Summary tab

When I finish the guided experience

Then the system shall create the appropriate Contract Payment Agreement & Preventive Care Agreement records as specified in the solution.

## Solution

Follow solution as described in [https://salesforce.atlassian.net/browse/PR1050558194|https://salesforce.atlassian.net/browse/PR1050558194|smart-link]

Reuse all the components listed there, make sure all components are parameterized to query the required data.
