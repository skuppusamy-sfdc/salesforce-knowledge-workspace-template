## Description

Steps to Reproduce:

1-Login as MSL User and navigate to Clinician Account record.

2-Navigate to Existing HPF Record.

3-Ensure status on HPF is Prospect.

4-Now without updating contracting status field update the Hospital Privileged Date and save it.

Issue  If on HPF record status is Prospect and without updating the Contracting Status, user update the Hospital Privileged Date then upon saving the record  Status is going from Prospect to Waiting to be Scheduled, it is skipping the Candidate status.

Expected Result - Ideally, it should not skip the Candidate status.

Actual Result - As of now it is skipping the Candidate Status and navigating to Waiting to be Scheduled.
