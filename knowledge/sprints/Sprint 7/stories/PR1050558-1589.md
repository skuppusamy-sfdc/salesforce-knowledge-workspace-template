## Description

As an Integration User, I need to know when First Scheduled Date is populated/updated in Salesforce, so that I can update downstream systems

## Acceptance Criteria

Given I am an Integration User

When a Healthcare Practitioner Facility record is updated with a new value for First Scheduled Date

Then a platform event is triggered with the following information: 

- Platform Event Fields, Name = HPFUpdates

** HPF Id

** Global Provider Id

** Global Facility Id

** First Scheduled Date

** Type = “First Scheduled Date”

## Solution

Create a Platform Even Name = HPFUpdates

Fields

1. HPFId = HealthcarePractitionerFacility.Id

1. GlobalProviderId = HealthcarePractitionerFacility.PractitionerId.Global_Provider_ID__pc

1. GlobalFacilityId = HealthcarePractitionerFacility.AccountId.Global_Facility_ID__c

1. FirstScheduledDate = HealthcarePractitionerFacility.First_Scheduled_Date__c

1. Type = “First Scheduled Date”

Create/Update a Record Triggered flow and when the HealthcarePractitionerFacility.First_Scheduled_Date__c is updated then send the platform event 

Security Matrix permission: **Data Integration**, **System Administrator**
