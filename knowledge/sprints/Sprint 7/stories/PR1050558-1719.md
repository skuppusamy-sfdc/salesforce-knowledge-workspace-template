## Description

As an MSL, I need Healthcare Practitioner Facility records to have accurate status values, so that downstream systems flow correctly and SCP corporate support users are viewing accurate information

## Acceptance Criteria

Given I am an MSL in the Corporate Support apps

When the following updates occur

Then the same Healthcare Practitioner Facility is updated as specified below: 

- Status: Prospect to Candidate

** When the Contracting Status field on an HPF is updated to Executed, update the Status to Candidate

- Credentialing Status: null to Pending

** When the Hospital Application Sent is populated (via integration from QGenda), update Credentialling Status to Pending

- Status: Candidate to Waiting to be Scheduled

** When the Hospital Privileged Date is populated (via integration from QGenda), update the Status to Waiting to be Scheduled

** _If First Clinical Shift is also populated at time of Hospital Privileged Date populated, go straight to Active status (see rows below)_

- Credentialing Status: Pending to Active

** When Location and Appointments Edit Detail in QGenda is updated (SF team to update with SF field), update Credentialing Status to Active

** Temp Inactive reason in QGenda is updated, update Temp Inactive reason to match

- Status: Candidate OR Waiting to be Scheduled to Active

** When the First Clinical Shift field on an HPF is populated (via integration from QGenda), update the Status to Active

** _Note this is built in_ [https://salesforce.atlassian.net/browse/PR10505581716|https://salesforce.atlassian.net/browse/PR10505581716|smart-link] _and is only included in this story for complete view of status updates_

- _Credentialling Status and Temporary Inactive Reason are populated in Salesforce from QGenda data via integrations_

## Solution

Record triggered flow on HPF, with the following conditional updates

IF HPF.Contract_Status = “Executed”    THEN  HPF.Status = Candidate

IF HPF.Hospital_Application_Sent_Date__c is populated (blank to value) THEN HPF.Credentialling_Status__c = Pending

IF HPF.Hospital_Privileged_Date__c    is populated (blank to value), HPF.Status = Waiting to be Scheduled

IF HPF.First_Clinical_Shift__c is populated, HPF.Status__c = Active
