## Description

As a Legal team member, I need to access to a Tactical Team queue for approvals, so that approvals can be associated to a queue.

## Acceptance Criteria

Given that I am a **Tactical Team queue member** in the Corporate Support or Corporate Support Console App

When I navigate to the Approval Work Items tab 

Then I can view Approval Work Item that have been assigned to the **Tactical Team Queue** And **Status = Assigned**

Then I can see the following columns:  Related Record, Related Record Object Name, Approval Condition Name, Created Date, Status, Assigned To, Approval Work Item Name

_Note: queue members are not part of a specific persona - membership is managed for specific individuals by assigning their user record to the queue_

## Solution

Use OOTB Feature

!image20250810143347.png|width=1192,height=234,alt="image20250810143347.png"!

** New List view = **Tactical Team Work Items*

** Filter by: Assigned To = Tactical Team Queue & Status = Assigned

** Columns: Related Record, Related Record Object Name, Approval Condition Name, Created Date, Status, Assigned To, Approval Work Item Name

 Testing: when you assign to a specific user, then Approval Work Item should be visible in Assigned Approval Work Items & My Outstanding Approvals (unsure if this got fixed).
