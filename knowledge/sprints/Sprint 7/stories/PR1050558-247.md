## Description

## Solution

Create Decision Matrix that has the following columns

Name: Task Creation Decision

CaseSubject (Input, Text)

TaskSubject(Output, Text)

RelatedToType (Output, Text)

RelatedTo (output, Text)

AssignedToType (Output, Text)

AssignedTo (Output, Text)

Record triggered flow on Case, to create tasks (related to case) based on the Decision matrix created above, where the input is Case subject and output of the Decision matrix can be used to create task records.

Task record type: Default
