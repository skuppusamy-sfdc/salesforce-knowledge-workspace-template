## Description

As a Corporate Support user, I need to easily see an indicator if a clinician has a Clinical Match Group value of 2, 3, or 4, so that I can quickly understand urgent information to answer calls appropriately

## Acceptance Criteria

Given that I am a Corporate Support user in the Corporate Support app

When I open a clinician person account record where **Clinical Match Group = 4**

Then I see a grey box displayed below the header and above the tabs component with the following message: 

DECEASED CLINICIAN

If you believe this is an error, please contact Salesforce Data Quality at [DL.SalesForceDQ@scphealth.com|mailto:DL.SalesForceDQ@scphealth.com]

Given that I am a Corporate Support user in the Corporate Support app

When I open a clinician person account record where **Clinical Match Group = 3**

Then I see a red box displayed below the header and above the tabs component with the following message: 

APPROVAL REQUIRED

Contact Risk and Legal at RM@SCPHealth.com

If this provider calls you, state the following: _“I see that you have been in the system previously. Let me pull your file, review it with the appropriate personnel and call you back soon.”_

Given that I am a Corporate Support user in the Corporate Support app

When I open a clinician person account record where **Clinical Match Group = 2**

Then I see an orange box displayed below the header and above the tabs component with the following message: 

APPROVAL REQUIRED

Contact Risk and Legal at [RM@SCPHealth.com|mailto:RM@SCPHealth.com]

If this provider calls you, state the following: _“I see that you have been in the system previously. Let me pull your file, review it with the appropriate personnel and call you back soon.”_

## Solution

3 rich text components on Clinical Person Account lightning record page dynamically displayed based on Clinical Match Group field value

Create three rich text component on lightning record page.

**Rich text 1**

**Label** : Clinical Match Group 4 Text

**Content**:

DECEASED CLINICIAN

If believe this is an error, please contact Salesforce Data Quality at [DL.SalesForceDQ@scphealth.com|mailto:DL.SalesForceDQ@scphealth.com]

**Background** **color**: grey

**Placement**: top of the right section and above the tabs component

**Component** **Visibility**: Clinical Match Group = 4

**Rich text 2**

**Label** : Clinical Match Group 3 Text

**Content:**

Contact Risk and Legal at [RM@SCPHealth.com|mailto:RM@SCPHealth.com]

**Background color:** Red

**Component** **Visibility**: Clinical Match Group = 3
