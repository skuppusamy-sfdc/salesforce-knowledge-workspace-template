## Description

As a Clinician Contracting Specialist, I need to view clinician contract and rate records, so that I can generate agreements with accurate information

## Acceptance Criteria

Given I am a Clinician Contracting Specialist in the Corporate Support apps

When I navigate to the Contracts tab

Then I can see the following list views: 

- All Contracts

** Filters: none

** Fields to display: Contract Number, Account Name, Status, Contract Start Date, Contract End Date, Contract Record Type, Contract Owner Alias

- All Client Contracts

** Filters: Contract Record Type = Client

** Fields to display: Contract Number, Account Name, Status, Contract Start Date, Contract End Date, Contract Owner Alias

- All Clinician Contracts

** Filters: Contract Record Type = Clinician

** Fields to display: Contract Number, Account Name, Status, Contract Start Date, Contract End Date, Contract Owner Alias

- All CPA Contracts

** Filters: Contract Record Type = Collaborative Practice Agreement

** Fields to display: Contract Number, Account Name, Status, Contract Start Date, Contract End Date, Contract Owner Alias

- My Client Contracts 

** Owner = My Contracts

** Fields to display: Contract Number, Account Name, Status, Contract Start Date, Contract End Date, Contract Record Type

- All Clinician Contracts - Approved

** Filters: Contract Record Type = Clinician, Status = Approved

** Fields to display: Contract Number, Account Name, Employment Type, Status, Contract Start Date, Contract End Date, Contract Approval Date, Contract Owner Alias

- Unassigned Clinician Contracts  Approved- (moved to 1809)

** Filters: Contract Record Type = Clinician, Status = Approved, Owner = Clinician Contracting Queue

** Fields to display: Contract Number, Account Name, Employment Type, Status, Contract Start Date, Contract End Date, Approval Date, Contract Owner

- My Clinician Contracts - Approved

** Filters: Contract Record Type = Clinician, Status = Approved, Owner = My Contracts

** Fields to display: Contract Number, Account Name, Employment Type, Status, Contract Start Date, Contract End Date, Contract Approval Date

- _Recently Viewed (standard list view)_
