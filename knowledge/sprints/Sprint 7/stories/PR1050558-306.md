## Description

As a Medical Staff Liaison, I need to terminate a clinician’s employment at a specific facility in Salesforce, so that I can accurately update the status of employees at facilities that SCP staffs for

## Acceptance Criteria

Given I am a Medical Staff Liaison in the Corporate Support apps

When I view a **Clinician** record

Then I am able to select Initiate Termination Request and see the following: 

- Facility Selection  [https://salesforce.atlassian.net/browse/PR1050558701|https://salesforce.atlassian.net/browse/PR1050558701|smartlink] 

- Termination Details

** Answer each question below per facility (when HPF status is not Prospect or Inactive AND Employment Type is Employee or Independent Contractor):

*** Last Day (date, required)

**** Help Text: Agreement termination date

*** Date Termination Received (date, required)

**** Help Text: Date termination request was received

*** Termination Request Date (date, automatically populated to today)

**** Help Text: Date termination request was submitted or automated

*** Voluntary Termination (checkbox, optional)

*** Reason for Termination (picklist dependent on Voluntary Termination, required)

**** picklist values to be provided - must align with Workday and Asymbl (why an open req was created due to clinician exit)

*** Exit Interview Requirement (checkbox, optional)

*** Bonus Repayment (checkbox, optional)

*** Ability to attach files

## Solution

For the selected HPF from previous screen (701) 

Display the fields mentioned above as a edit block, edit view with the help icons at the table header

on completion, create Case records of record type Offboarding (added to Perm Set = **Manage Clinician Onboarding**)

Needs to include additional fields listed in story, it was not added/available when 1775 was implemented. 

NOTE:

in the first screen, we show the all HPFs (that are not inactive)

User selects the HPF

Amongst the selected HPF, we apply the filter condition (when HPF status is not Prospect or Inactive AND Employment Type is Employee or Independent Contractor) and show only those HPFs which satisfy the condition.

Question

Should these fields be created in DD on the Case object
