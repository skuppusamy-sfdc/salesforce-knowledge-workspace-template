## Description

As a Corporate Support team member, I need to access to a Corporate Entity Mismatch Queue for approvals, so that approvals can be associated to a queue

## Acceptance Criteria

Given that I am a **Corporate Entity Mismatch queue member** in the Corporate Support or Corporate Support Console App

When I navigate to the Approval Work Items tab 

Then I can view Approval Work Items that have been assigned to the **Corporate Entity Mismatch** Queue And **Status = Assigned**

Then I can see the following columns:  Related Record, Related Record Object Name, Approval Condition Name, Created Date, Status, Assigned To, Approval Work Item Name

And in the OOTB  list view = Assigned Approval Work Items, I can see “Assigned To” between “Status” and “Approval Work Item Name”

_Note: queue members are not part of a specific persona - membership is managed for specific individuals by assigning their user record to the queue_
