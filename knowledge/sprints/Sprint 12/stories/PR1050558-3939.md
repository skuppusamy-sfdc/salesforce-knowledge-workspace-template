## Description

As a Clinician, I need to easily view data in the HC portal, so that I can quickly understand key information

## Acceptance Criteria

Given I am a clinician logged in to the HC portal

When I am on the **Home** page

Then I am able to see the following: 

- Scale down the sizing of the legend area (see wireframe for example)

- Add the following user greeting “Hello, [Salutation] [FirstName] [LastName]”

- Add Overall Status indication below user greeting and above active applications

- Active Applications, Onboarding, and Facilities are organized by Facility rather than Job

- In the Active Applications section

** Change “Job Application In Progress” column header to “Application”

** Change “Offer In Process” column header to “Offer”

- In the Onboarding section

** Internal

*** Remove “Internal” text above table

*** Chance “SCP Onboarding” column header to “SCP Enablement”

** Facility

*** Remove “Facility” text above the table

*** Remove “Facility Onboarding -” when displaying data in the Name column, only display the Account name

- Remove the blank Helpful Documents section

- Display the Global Provider ID field below the “Log in to mySCP” button and above “Helpful Links”

** Commas do not display

** If global ID does not exist display the following: “Global ID: n/a”

- Tasks component has the following columns: Subject, Due Date, Status, Priority, Last Modified Date

When I am on the **My Applications** page

Then I am able to see the following: 

- Applications are organized by Facility rather than by Job

- Facility is the blue header for each card

- Each card displays the following information below Facility: Job (no longer using Job Name), Initial Application Date, Recruiter, Recruiter Phone, Recruiter Email

- When in the “Withdraw Application” guided experience, confirmation has the following text: Are you sure you want to withdraw your application to [Job] at [Facility]? This action cannot be undone.
