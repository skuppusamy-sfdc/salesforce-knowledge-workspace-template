## Description

As a Clinician Contracting Specialist, I need IC NP/PA Contracts to terminate automatically on the anniversary date, so that clinician date is accurate and matches contract language

## Acceptance Criteria

Given I am a Clinician Contracting Specialist in the corporate support apps

When I navigate to a clinician contract record where:

- Employment Type = Independent Contractor

- Contract is associated with HPF records where Facility Title = Nurse Practitioner and/or Physician Assistant

Then when a year has passed from the Contract Start Date (Contract End Date = today’s date) AND no new clinician contracts in Status = Executed AND  Employment Type = Independent Contractor have been created

-Then look for Contract records that meet the following criteria:-

- Employment Type = Independent Contractor

- Start Date is after the Start Date of the original contract (where a year has passed)

Then the IC Contract record is automatically updated with the following: 

- Contract record updates

** Status = Canceled

** Cancel Reason = Terminated

And the system shall automatically create the Clinician Offboarding **Facility** cases & tasks for the Facility on the IC contract (populate Clinician, Service Account, and HPF lookups)

And system shall automatically create the Clinician Offboarding **SCP** cases & tasks if the Clinician has no other Active HPFs (Status ≠ Inactive)

And the system shall automatically close Facility and SCP (if applicable):

- Tasks with ‘Status’ = Canceled

- Cases with ‘Status’ = Closed, ‘Closed Reason’ = Complete

And the system shall automatically update the following:

- HPF record updates shall only be for the applicable IC NPPA HPF record(s)

** Status = Inactive

** Effective To = date the automation ran

** Termination Date = date the automation ran
