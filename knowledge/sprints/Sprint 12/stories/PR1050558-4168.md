## Description

**ISSUE 1** 

### **Description**

When viewing a **Letter of Intent Contract** with **Status = Executed**, the user is incorrectly able to **edit fields in the Contract Overview section and the Letter of Intent Request Info section**.

This violates Acceptance Criteria which states that these fields should be **read-only** once LOI Status is not Draft or Submitted.

### **Expected Behavior**

For LOI Contract **Status = Executed**:

** Contract Overview section fields → **Read Only*

** LOI Request Info section (Left column) → **Read Only*

** LOI Request Info section (Right column) → **Read Only*

- Only Key Dates section (limited fields) should remain editable.

### **Actual Behavior**

** User is able to **edit* Contract Overview fields in Executed status.

** User is able to **edit* LOI Request Info (Left & Right column) fields in Executed status.

### **Steps to Reproduce**

1. Log in as a **Legal Team Member**.

1. Navigate to **Corporate Support App → Contracts tab**.

1. Open an LOI Contract where **Status = Executed**.

1. Click **Edit**.

1. Observe that fields in Contract Overview + LOI Request Info are still editable.

### **Impact**

Legal team may unintentionally modify finalized LOI data, causing compliance and documentation risks.

**ISSUE 2** 

### **Description**

The field **Source Opportunity** is editable in **all LOI Status values**, including Executed, Draft, and Submitted.

This violates AC2 which states:

{quote}“Source Opportunity should NOT be editable in any status.”{quote}

### **Expected Behavior**

**Source Opportunity must be NON-editable** at all times (all statuses).

### **Actual Behavior**

** Source Opportunity remains **editable* across all statuses (Draft, Submitted, Executed).

### **Steps to Reproduce**

1. Log in as **Legal Team Member**.

1. Open any Letter of Intent Contract.

1. Click **Edit**.

1. Observe that **Source Opportunity** field is editable regardless of status.

### **Impact**

Incorrect linkage between Opportunity → LOI Contract may occur, creating data integrity issues.
