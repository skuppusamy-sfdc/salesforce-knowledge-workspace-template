## Solution

### **Create a Platform Event:** 

- Name: Clone Record

- API Name: Clone_Record__e

- Field

** Id of Record To Clone

** sObject Name

** Attribute 1

** Attribute 2

** Attribute 3

### Create Platform Event:

****Remove the Existing Logic to Create the Job Record for** [https://salesforce.atlassian.net/browse/PR10505582472|https://salesforce.atlassian.net/browse/PR10505582472|smart-link] ****

Case RT: Clinician Offboarding

When Case.Backfill_Recommended__c is updated and set as True and Healthcare_Practitioner_Facility__r.Job_Applicant__r.bpats__Job__c is not NULL

Create the Platform Event: Clone Record

- Row Id of Record To Clone = Healthcare_Practitioner_Facility__r.Job_Applicant__r.bpats__Job__c

- sObject Name = bpats__Job__c

- Attribute 1 = If Case.Voluntary_Termination__c = TRUE, Attrition  Resignation, Else Attrition  Termination

### Create Job Record:

Create a Platform Triggered Flow 

Setup Run This Flow as: Default Workflow User

Assign Permission to Default Workflow User: Mulesoft Asymbl Integration

When sObject Name = bpats__Job__c Then

 **CLONE** **ALL Field** the bpats__Job__c record Id = PlatformEvent.Id of Record To Clone

Set fields

- bpats__Job__c.bpats__Status__c = Draft

- bpats__Job__c.bpats__Post_Externally__c = FALSE

- bpats__Job__c.bpats__Posted_Date__c = null

- bpats__Job__c.Vacancy_Reason__c = PlatformEvent.Attribute 1

SendEmail to VPO as per AC from Platform Event Flow
