## Description

### **Description:**

When a **FINAL SUN email** is triggered, the **Client Start Up Time** in the email displays a different format than the **Client Start Up Time** shown on the **Provider Network Contract (PNC)** record in Salesforce.

The time value should match exactly between the email and the Salesforce record.

### **Steps to Reproduce:**

1. Log in to **Salesforce QA Sandbox**.

1. Navigate to the **Provider Network Contract (PNC)** record.

1. Note the **Client Start Up Time** (e.g., `12:15:00 AM`).

1. Trigger a **FINAL SUN email** by creating or updating a `SUN - Final` PMT Case.

1. Check the **email received** in Outlook/Gmail.

1. Compare the **Client Start Up Time** value in the email with the one on the PNC record.

### **Actual Result:**

** The **PNC record* displays time as: `12:15:00 AM`

** The **FINAL SUN email* displays time as: `00:15:00.000Z`

- The formats do not match.

### **Expected Result:**

The **Client Start Up Time** displayed in the **FINAL SUN email** should match the format used in Salesforce (`hh:mm:ss AM/PM`).
