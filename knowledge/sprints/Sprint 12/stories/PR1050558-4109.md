## Description

As a Corporate Support user with access to Client Contracts, when looking at a Client Contract with Contract Type ≠ Master, I want to see a message directing me back to the Master contract so that I can view the details of the whole bundle (Master and child contracts).

As a Corporate Support user with access to Letter of Intent Contracts, I want to see only valid Actions so that I do not accidentally select an invalid action.

## Solution

Update the Record Page for Contract.RT = Client Contract

All the changes is handled via Record page by using conditional filters with duplicate components if needed 

**Highlight Panel** → Create 2 highlights panel and show based on Contract.Contract_Type__c = Master  and one for Contract.Contract_Type__c != Master  as per AC

**Tabs = Details** ->

- Contract Type ≠ Master  As per AC show for

- Contract Type = Master no change

**Tab = Provider Network Contracts**: → Show based on condition Contract.Contract_Type__c = Master  as per AC

**Tab = Related** → Show based on condition Contract.Contract_Type__c = Master as per AC
