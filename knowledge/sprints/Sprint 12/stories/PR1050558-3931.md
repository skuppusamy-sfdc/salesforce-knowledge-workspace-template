## Description

As a System Administrator, I was to refine HPF color logic when the user is viewing a Clinician Contract’s Summary tab, so that we simplify the logic and ensure we are supporting both new and cloned contracts in a unified manner.

## Acceptance Criteria

Tech story to refine the logic to change the colors of HPFs in the Summary tab. Original story [+PR10505581389+|https://salesforce.atlassian.net/browse/PR10505581389]

AC1: White/No color

Given I am a corporate support user with access to Clinician Contracts

When I am viewing a Clinician Contract’s **Summary** tab

And an HPF:

- Is NOT marked as deleted

- Is cloned 

Then HPFs shall be displayed in white/no color

AC2: Gray color

Given I am a corporate support user with access to Clinician Contracts

When I am viewing a Clinician Contract’s **Summary** tab

And an HPF:

- Is marked as deleted

- Is cloned 

Then HPFs shall be displayed in Gray

AC3: Green color

Given I am a corporate support user with access to Clinician Contracts

When I am viewing a Clinician Contract’s **Summary** tab

And an HPF:

- Is NOT marked as deleted

- Is NOT cloned 

Then HPFs shall be displayed in Green

And if this is a new contract, then HPFs will be displayed in green as well.
