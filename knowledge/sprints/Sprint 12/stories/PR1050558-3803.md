## Description

As a legal team member, when a Client Offboarding parent case is created for a Service Account, I want the ability view related change cases, so that we can track and ensure appropriate offboarding activities have been completed.

## Acceptance Criteria

AC:

Given I am a Legal Team member (or any user with access to view Client Offboarding case record type)

When I am on a Case (RT: Client Offboarding)

Then I have the ability to view tab = “Related”

And in the “Related” tab, I can see a related Cases related list, where users can view any child cases

- Fields to display: Subject, Priority, Status

## Solution

Add Related Cases related list based on [Parent Case|https://corehc-dev1.sandbox.my.salesforcesetup.com/lightning/setup/ObjectManager/Case/FieldsAndRelationships/Parent/view]
