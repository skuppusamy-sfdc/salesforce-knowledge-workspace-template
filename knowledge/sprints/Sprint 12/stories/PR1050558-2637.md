## Description

As a Legal Team member, when the related tasks to terminate a facility (Service Account) are manually closed by the task assignee, I want the child and parent Client Offboarding cases to be automatically closed so that I can track the timely termination of a facility (Service Account).

## Acceptance Criteria

AC1: Auto Close Child Case - Closed /Complete

Given I am a Legal Team member in the Corporate Support apps

**When all of the Task records related to a Client Offboarding Child Case are Closed (Status = Completed or Canceled)**

And the child Case record does not already have Status = Closed

Then the child Case is automatically updated to:

- Status = Closed

- Closed Reason = Complete

-AC2: Auto Close Child Case  Closed /Canceled [moved to story 1684]

-Given I am a Legal Team member in the Corporate Support apps-

**When there are NO Task records related to a Client Offboarding Child Case**

-And the child Case record does not already have Status = Closed-

-Then the child Case is automatically updated to:-

- Status = Closed

- Closed Reason = Canceled

AC3: Auto Close Parent Case

Given I am a Legal Team member in the Corporate Support apps

**When all of the child Case records related to a Client Offboarding Parent Case are Status = Closed & Closed Reason = Completed or Canceled)**

And the Parent Case record does not already have Status = Closed

Then the Parent Case is automatically updated to:

- Status = Closed

- Closed Reason = Complete

NOTE: added AC2 since we had an open action item on 1684, we are unsure all the which cases to create or not create in story 1683. - If we can align on 1684 (tasks), then we can add an enhancement story to just NOT create the child case based on the service line, rendering AC2 obsolete.   

NOTE: if the user **manually adds related tasks to the Parent Case**, those task will also need to be closed by the user for the system to auto close the Parent Case.

## Solution

Check if Flow Close Related Cases is already covering the Case Close if not update the Flow
