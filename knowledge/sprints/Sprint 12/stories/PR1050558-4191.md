## Description

As a System Administrator, when a Clinician leaves SCP (deactivated) and decides to reapply/rehire, I want the system to reactivate the Clinician’s previous Experience Cloud User record instead of creating a new user record, so that we keep our system clean.

As a System Administrator, I want to ensure that when a clinician’s user has the appropriate permission, the system can direct them to the appropriate sign-in method.
