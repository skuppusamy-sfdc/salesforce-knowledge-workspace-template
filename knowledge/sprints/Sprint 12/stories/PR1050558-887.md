## Solution

Query 1: SOQL Select Services_Sold__c, Leased_Employee__c, If_Leased_Employees_Ins_Provided_By__c, Client_Statup_Date__c, Facility_Start_Up_Time__c, Client_Start_Up_Time_Zone__c, ContractId FROM  ProviderNetworkContract WHERE ProviderNetworkContract.Contract.Contract_Type__c = “Master” AND ProviderNetworkContract.Contract.SourceOpportunityId = inov8__PMT_Phase__r.inov8__Project__r.Opportunity__c AND ProviderNetworkContract**.Service_Account__c** = inov8__PMT_Phase__c.Service_Account__c AND ProviderNetworkContract.Contract.Agreement_Category__c = “Facility Agreement” Order By ProviderNetworkContract.Contract.StartDate DESC LIMIT 1

–> ContractId in Query 1 is the Master Contract Id (used as ParentContractId in all Child contract records)

Query 2: Select RelatedAccount.Name, PartyRoleRelation.RoleName FROM AccountAccountRelation WHERE **Realted**AccountId = inov8__PMT_Phase__c.Service_Account__c AND PartyRoleRelationId = (Select Id from PartyRoleRelation where RoleName = “Client Owner” OR RoleName = “Services Contracting Entity” ) AND StartDate <= Today() AND (EndDate IS NOT NULL OR EndDate >= Today()) Order By StartDate DESC  LIMIT 1)

Query 3: Select ED_Volume__c,  HM_Volume__c, TeleMedicine__c, Insurance_Limits_per_Occurrence__c, Insurance_Limits_Aggregate__c FROM Pro_Forma_Facility_Detail__c WHERE Pro_Forma_Facility_Detail__c.Model__r.Opportunity__c = inov8__PMT_Phase__r.inov8__Project__r.Opportunity__c AND Pro_Forma_Facility_Detail__c.Model__r.Status__c =  “Approved for Contracting” Order BY Created DESC LIMIT 1)

-Query 4: SOQL Select Occurrence_Limit__c, Annual_Aggregate__c, Leased_Employee__c FROM  ProviderNetworkContract WHERE ProviderNetworkContract.Contract.Contract_Type__c = “Insurance” AND ProviderNetworkContract.Contract.SourceOpportunityId = inov8__PMT_Phase__r.inov8__Project__r.Opportunity__c AND ProviderNetworkContract.**Service_Account__c** = inov8__PMT_Phase__c.Service_Account__c AND ProviderNetworkContract.Contract.ParentContractId = Query1.ContractId AND ProviderNetworkContract.Contract.Agreement_Category__c = “Facility Agreement” Order By ProviderNetworkContract.Contract.StartDate DESC LIMIT 1-
