## Root Cause

The system is successfully identifying and retrieving the **Facility Global ID** (sourced from the active HPFs, such as those associated with Global Provider ID 46923). However, the subsequent **lookup process** that uses this ID to fetch the corresponding **EMR system details** is failing. This prevents the EMR information from being displayed, resulting in the "unavailable" message.
