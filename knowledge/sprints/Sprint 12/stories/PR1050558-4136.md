## Description

Issue - Upon completion of Amend Contract the new LOI Contract should be displayed and pop. up of completion shouldn't display.

Expected Result - Upon completion of Amend Contract the new LOI Contract should be displayed and pop. up of completion shouldn't display.

Actual Result - Upon completion of Amend Contract the new LOI Contract is not displayed and pop. up of completion is displayed.

Steps to Reproduce:-

1-Login as Legal Team member into the Core HC org and navigate to the Corporate Support/Console Apps.

2-Navigate to Contracts tab → Open record type having Letter of Intent

3-Verify the Amend Contract guided experience option is available.

4-Select the “Amend Contract” guided experience.

5-Populate the fields and click on Save.

## Root Cause

The expected behaviour is to send the user to the newly created record, but as the implementation is done on screen flow and to make it consistent we have implemented the POP-up. Later the existing implementation has been approved.

As per reconfirmation from Sakthi, the original implementation remains as it is.

User Interface: Inconsistent UI Elements
